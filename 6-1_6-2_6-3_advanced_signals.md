# 6.1 연결재무제표와 지배구조: 숨겨진 레버리지를 찾아라

*— 자회사 부채는 모회사 재무제표에 보이지 않을 수 있다*

---

### ① 시그널 도입부

그룹사를 분석할 때 어떤 재무제표를 기준으로 해야 하는가.

연결이냐, 별도냐. 이것이 전략적 선택이다.

지주회사 구조에서 별도 재무제표만 보면 그룹 부채가 보이지 않는다. 자회사의 손익이 배당금으로만 들어온다. 자회사가 쓰러지기 직전까지 모회사 별도 재무제표는 멀쩡하게 보인다.

반대로, 사업 자회사 분석에서 연결만 보면 지주회사나 다른 계열사의 영향이 섞여서 순수한 사업 수익성이 흐려진다.

핵심 원칙은 하나다.

**경제적 실질을 기준으로 선택하라.**

그 기업이 실제로 통제하고 있는 경제적 단위가 무엇인가. 그것이 연결 범위다. 그 범위에서 벗어나는 부채와 리스크를 주석에서 찾는 것이 연결 분석의 완성이다.

이 원칙이 실전에서 어떤 차이를 만드는지 확인하자.

---

### ② 알고리즘 설명

#### 연결의 기본 원리

```
연결 재무제표 = 모회사 + 자회사 전체를 하나의 경제적 실체로 합산
              - 내부거래 제거 (그룹 내 거래는 외부에서 보면 없는 거래)

내부거래 제거:
  모회사가 자회사에 물건을 팔면 → 연결에서 그 매출과 매출원가는 제거
  자회사가 모회사에 자금을 빌려주면 → 연결에서 채권채무 상계
```

**비지배지분(NCI)**: 100% 이하 자회사의 경우 나머지 지분을 가진 외부 주주 몫.

```
연결 당기순이익 = 지배주주 귀속 순이익 + NCI 귀속 순이익
주주가치 = 지배주주 귀속 순이익 / 발행주식수 = 연결 EPS
```

#### K-IFRS 10의 핵심: 실질 지배력

K-IFRS 10은 의결권 50% 이상이 아니라 **실질 지배력**을 연결 기준으로 한다.

의결권이 45%여도 이사회를 지배하고 경영진을 임면한다면 연결 대상이다. 반대로 50% 초과 지분이 있어도 실질적으로 다른 주체가 지배하면 연결 제외될 수 있다.

투자자 입장에서 중요한 것: 의결권 구조만으로 연결 범위를 판단하지 마라. 주석의 연결 범위 설명을 확인해야 한다.

---

### ③ 📋 케이스 스터디 — "내부거래로 만든 이익"

**코스피 그룹사 P사 | 특수관계자 매출 의존도 분석**

---

**숫자**

| 항목 | 연결 기준 |
|---|---|
| 연결 매출 | 12,400억 |
| **특수관계자 매출** | **7,500억 (61%)** |
| 외부 고객 매출 | 4,900억 (39%) |
| 연결 영업이익 | 620억 (OPM 5.0%) |
| **외부 고객 매출 기준 추정 OPM** | **약 2~3%** |

연결 매출의 61%가 그룹 내 특수관계자 거래였다.

---

**해부**

P사는 그룹 계열사들에게 IT 서비스와 물류 서비스를 제공했다. 계열사 거래는 외부 경쟁 없이 이루어졌다. 계약 가격이 시장 가격보다 높게 설정될 가능성이 있었다.

만약 계열사 거래가 없어진다면:
- 매출이 39% 수준으로 급감
- 고정비 커버가 어려워져 영업이익 급감
- 실질 수익성이 현재 5%에서 훨씬 낮아질 것

P사의 높은 이익률이 독립적인 경쟁 우위에서 나온 것인지, 그룹 내 거래의 특혜에서 나온 것인지 — 이 구분이 P사의 내재가치를 결정한다.

---

**결말**

2년 후, 그룹 구조조정이 시작됐다. 지주회사가 물류 부문 계열사를 외부에 매각했다. P사에 대한 물류 서비스 용역 계약이 종료됐다. 연간 매출 2,800억이 사라졌다.

P사의 매출은 12,400억에서 9,200억으로 -26% 급감했다. 그러나 타격은 매출 감소보다 더 심각했다. 계열사 거래에서 높은 마진을 받고 있었기 때문에, 영업이익은 620억에서 140억으로 -77% 급감했다. 외부 고객 기준 OPM은 1.5%에 불과했다. 계열사의 특혜가 아닌 독립적 경쟁력은 처음부터 거의 없었던 것이다.

주가는 구조조정 공시 이후 6개월간 -39% 하락했다.

---

**교훈**

> [특수관계자 거래 주석]에서 매출 의존도를 계산하라.
> 특수관계자 매출 비중이 50% 초과하면 —
> 외부 경쟁력만으로 이 이익을 낼 수 있는지 의문을 품어라.
> 그룹 해체나 계열사 관계 변화가 이 기업에 미치는 영향을 시뮬레이션하라.
> P사의 투자자들은 매출의 61%가 "계약"이 아니라 "관계"에 달려 있었다는 것을 뒤늦게 알았다.

---

| 구분 | 긍정(+) 시그널 | 부정(-) 시그널 |
|---|---|---|
| **연결 vs 별도 부채비율** | 차이 50%p 미만 | 차이 100%p 초과 |

### ⑤ DART 실전 박스

```
Step 1. DART에서 두 보고서 나란히: 연결 vs 별도

Step 2. 주석 → [특수관계자 거래]

Step 3. 주석 → [종속기업 현황]
        연결 범위 확인, 의외의 자회사 또는 제외된 기업 파악
```

### 📊 내재가치 체크리스트

```
별도 부채비율:  (    )%  vs  연결 부채비율: (    )%
차이:          (    )%p

특수관계자 매출 / 연결 매출 = (    )%
→ 50% 초과: 외부 경쟁력 단독 평가 필요

NCI 귀속 순이익: 흑자 / 적자

```

# 6.2 외화환산: 환율이 이익을 어떻게 왜곡하는가

*— 글로벌 기업 분석의 환율 효과 분리*

---

### ① 시그널 도입부

2022년, 한 코스피 수출주의 영업이익이 전년 대비 +47% 급증했다. 증권사들은 목표가를 올렸다. "실적 서프라이즈." 주가는 연초 대비 +32% 올랐다.

그러나 Constant Currency 기준으로 재계산하면 영업이익 성장률은 **+11%**에 불과했다. 나머지 36%p는 원화 약세(원달러 1,180원 → 1,325원)가 만들어낸 착시였다. 환율이 실력을 가장한 것이다.

2024년, 원화가 다시 강세로 전환됐다. 원달러 1,325원 → 1,285원. 같은 기업의 영업이익이 -8% 감소했다. 시장은 실망했다. 주가는 -23% 하락했다. 그런데 Constant Currency 기준으로는 +14% 성장이었다. 실제 판매량과 ASP는 오히려 개선됐는데, 달러로 번 돈이 원화로 환산되면서 줄어든 것이었다.

2년간의 주가 변동을 보면 — 2022년에 시장이 환율을 실력으로 착각해서 올린 가격을, 2024년에 환율이 역전되면서 전부 토해냈다. 실질 사업 성과는 두 해 모두 양호했는데, 투자자는 환율의 롤러코스터를 탄 것이다.

이것이 수출 비중이 높은 기업을 분석할 때 환율 효과를 분리해야 하는 이유다.

환율은 기업 자체의 경쟁력과 무관하게 이익을 30% 이상 바꿀 수 있다. 이 효과를 제거하지 않으면 실력을 환경으로 착각하거나, 환경을 실력으로 착각한다.

---

### ② 알고리즘 설명

#### 두 가지 외화 이슈

**① 외화 거래손익 (Transaction Gain/Loss)**
달러로 거래하는 기업에서, 계약과 결제 사이 환율이 변하면 손익이 발생한다.
→ 손익계산서에 당기 이익으로 반영.

**② 해외 자회사 환산 (Translation)**
해외 자회사의 재무제표를 원화로 환산할 때 환율 변동 영향.

```
현행환율법 (기능통화 = 현지통화):
  자산·부채: 기말환율 적용
  손익: 평균환율 적용
  차이: OCI로 처리 (환산조정) → 손익계산서 통과 안 함

시제법 (기능통화 = 원화):
  화폐성 항목: 기말환율
  비화폐성 항목: 역사적환율
  차이: 당기 손익으로 반영 → 이익 변동성 큼
```

#### 수출주 분석: 환율 효과 분리

```
환율 효과 = 원화 기준 매출 - Constant Currency 기준 매출

Constant Currency 매출:
  달러 매출량 × 전기 적용 환율

환율 효과가 커질수록 → 이익의 환율 의존도 분석 필요
```

---

### ③ 📋 케이스 스터디 — "환율을 실력으로 착각한 2년"

**코스피 대형 전자기업 R사 | 2022~2024년 환율 효과 분리 분석**

---

**숫자 — 환율의 롤러코스터**

| 항목 | 2022년 | 2023년 | 2024년 |
|---|---|---|---|
| 원화 기준 영업이익 | 11,200억 | 8,200억 | 7,544억 |
| Constant Currency 기준 | 7,900억 | 8,200억 | 9,348억 |
| **환율 효과** | **+3,300억** | **0** | **-1,804억** |
| 평균 원달러 환율 | 1,290원 | 1,325원 | 1,285원 |
| 수출 비중 | 78% | 78% | 80% |
| **주가 변동 (연간)** | **+32%** | **-5%** | **-23%** |

2022년 원화 약세 시기, R사의 원화 기준 영업이익 11,200억 중 3,300억(29%)은 환율이 만들어낸 숫자였다. 시장은 이것을 "실적 서프라이즈"로 읽었다. 2024년 원화 강세로 전환되자, 같은 메커니즘이 역으로 작동했다. 실질 사업 성과는 Constant Currency 기준으로 7,900억 → 8,200억 → 9,348억으로 **매년 개선**되고 있었다. 그러나 주가는 환율의 방향을 따라갔다.

---

**해부**

R사의 원가는 대부분 원화 기반이었다 (국내 인건비 60%, 국내 부품 25%). 매출은 달러 기반 (수출 80%). 이 구조는 **통화 불일치(Currency Mismatch)** 의 전형이다.

원화 약세 → 달러 매출의 원화 환산액 증가 → 원가 불변 → 마진 확대
원화 강세 → 달러 매출의 원화 환산액 감소 → 원가 불변 → 마진 축소

자연헤지가 없는 구조에서 환율 3% 변동이 영업이익 22%p 변동으로 증폭된 것이다. 수출 비중이 높을수록 이 레버리지 효과가 커진다.

만약 R사가 원자재를 달러로 수입하는 구조였다면(원가도 달러), 환율이 매출과 원가를 동시에 움직여 영향이 상쇄됐을 것이다. 이것이 **자연헤지**다.

---

**결말**

2022년에 R사를 "실적 개선주"로 매수한 투자자는 2024년까지 누적 -19%의 손실을 기록했다. 반대로, Constant Currency 기준 실적이 매년 개선되고 있다는 점에 주목하고 2024년 주가 급락 시 매수한 투자자는 이후 원화 약세 반전과 함께 +28%의 수익을 거뒀다.

환율 효과를 분리할 줄 아는 투자자와 모르는 투자자 사이에 2년간 47%p의 수익 차이가 벌어졌다.

---

**교훈**

> 수출 비중이 높은 기업의 실적 발표 시,
> 환율 효과를 제거한 Constant Currency 기준 실적을 함께 확인하라.
> 원화 기준 실적만 보면 환율에 의한 왜곡을 사업 성과로 오해할 수 있다.
> R사의 2년은 증명한다 — 환율은 양방향으로 착시를 만든다.

---

### ④ 투자자 시그널 대조표

| 구분 | 분석 포인트 |
|---|---|
| **원화 강세 + 수출 비중 높음** | 실적 발표 전 환율 효과 선계산 |
| **환산조정 OCI 누적 마이너스** | 원화 강세 지속 시 자본 추가 감소 |
| **Constant Currency 공시** | 기업이 직접 제공하면 활용 |
| **자연헤지 구조** | 달러 매출 + 달러 원가 기업은 환율 영향 상쇄 |

---

### ⑤ DART 실전 박스

```
Step 1. 사업보고서 → [사업의 내용] → 수출 비중 확인
Step 2. 주석 → [외환위험 관리]
        통화별 매출, 비용 구성 확인
        환헤지 현황 (선물환, 통화스왑 등)
Step 3. 자본변동표 → 환산조정 OCI 누계 확인
Step 4. 사업보고서 → [환율 민감도 분석]
        원달러 100원 변동 시 이익 영향 확인
```

# 6.3 공시 분석 심화: 주석이 말하는 것들

*— 재무제표 본문보다 주석에서 더 많은 진실이 드러난다*

---

### ① 시그널 도입부

사업보고서를 여는 순서가 있다.

대부분의 투자자는 이렇게 연다. 손익계산서 → 재무상태표 → 끝.

분석가는 다르게 연다. 감사보고서(KAM) → 주석 1번(중요 회계정책) → 우발채무 → 특수관계자 → 차입금 만기 → 연금 → 그리고 나서 손익계산서.

주석을 먼저 읽는 이유는 하나다. 본문 숫자를 어떻게 해석해야 하는지를 주석이 결정하기 때문이다.

수익인식 정책을 모르고 매출 숫자를 보면 절반을 모르는 것이다. 차입약정 조건을 모르고 부채비율을 보면 위험을 모르는 것이다. KAM을 읽지 않으면 감사인이 어디서 긴장했는지를 모르는 것이다.

주석은 숫자의 설명서다. 설명서를 읽지 않고 기계를 작동시키면 어떻게 되는가.

---

### ② 알고리즘 설명

#### KAM: 감사인이 가장 긴장한 곳

핵심감사사항(Key Audit Matters)은 감사인이 당해 감사에서 가장 유의적인 판단이 필요했던 영역이다.

```
KAM이 자주 등장하는 항목:
  - 수익인식 (장기 계약, 진행기준)
  - 영업권 손상 (주관적 가정)
  - 금융자산 공정가치 (Level 3)
  - 재고자산 평가 (진부화)
  - 소송 충당부채 (결과 예측 불확실)
```

KAM에 등재됐다는 것은 감사인이 이 항목에서 경영진과 깊은 논의가 필요했다는 의미다. 그리고 그 논의의 결과가 재무제표에 반영됐다는 의미다.

KAM 항목은 투자자가 깊이 들여다봐야 할 주석의 지도다.

#### 후속사건: 공시 전 선행 정보

보고기간 후(결산일 이후 ~ 공시일 이전)에 발생한 중요 사건을 후속사건으로 공시한다.

```
인식 후속사건: 결산일 현재 이미 존재했던 상황이 사후에 확인됨
              → 재무제표 본문 수정
비인식 후속사건: 결산일 이후 새로 발생한 사건
               → 주석에만 기재
```

후속사건 주석은 공식 공시 전에 중요한 M&A, 소송 결과, 자금 조달 계획을 미리 알 수 있는 통로다. 분기보고서 제출 전에 사업보고서 주석을 확인하면 선행 정보를 얻는 경우가 있다.

#### Claude Code 자동추출 우선순위 6개

이 섹션에서 다루는 주석 항목들은 Claude Code DART 파싱 자동화의 핵심 타깃이다.

```
① 차입금 만기구조 → 리파이낸싱 리스크 자동 계산
② 우발부채 금액 → 기업가치 조정 입력값
③ 특수관계자 매출 비중 → 실질 외부매출 자동 계산
④ 개발비 자산화 금액 → 조정 OPM 계산
⑤ 리스부채 잔액 → 조정순부채 계산
⑥ KAM 항목 → HRM 정성 조정 트리거 자동 감지
```

이 6개 항목의 자동 추출이 완성되면 — DART Insight 딥재무평가 콘텐츠의 데이터 레이어가 완성된다.

---

### ③ 📋 케이스 스터디 — "KAM이 알려준 것"

**코스닥 IT 서비스 기업 S사 | KAM 경고 → 실제 손실 발생**

---

**그날의 감사보고서**

S사의 감사보고서를 열자 KAM이 두 개였다.

```
KAM 1: "수익인식 — 장기 프로젝트 완성도 측정의 불확실성"
  → 진행기준 적용 시 완성도 추정에 경영진 판단이 개입
  → 주요 프로젝트 5개 검토 결과 중 1개에서 추정 방법 이견

KAM 2: "매출채권 — 주요 거래처 회수 가능성 평가"
  → 3개 주요 거래처의 연체 현황 및 회수 가능성 불확실
  → 대손충당금 적정성에 유의적 판단 필요
```

감사의견은 적정이었다.

---

**숫자**

| 항목 | KAM 등재 시점 | 6개월 후 |
|---|---|---|
| 매출채권 90일 초과 비중 | 38% | 61% |
| 계약자산(미청구공사) | 3,200억 | 4,800억 |
| 영업이익 | 흑자 | **적자 전환** |
| 대손충당금 설정 | — | **90억 신규 설정** |

KAM이 경고했던 두 가지가 모두 현실화됐다.

---

**해부**

KAM은 재무제표가 옳다는 확인이 아니다. 감사인이 가장 주의를 기울인 영역의 표시다. 수익인식과 매출채권이 KAM에 동시에 올라왔다는 것은 이 두 항목에 불확실성이 집중돼 있다는 신호였다.

적정의견이 "이 기업이 안전하다"는 뜻이 아님을 다시 한번 확인한 사례다.

---

**교훈**

> 감사보고서에서 KAM 항목을 먼저 확인하라.
> KAM에 수익인식, 영업권 손상, 매출채권 회수가 있으면
> 해당 주석을 집중 분석하라.
> KAM은 감사인이 그린 위험 지도다.

---

### ④ 투자자 시그널 대조표

| 구분 | 긍정(+) 시그널 | 부정(-) 시그널 |
|---|---|---|
| **KAM 항목** | 일반적 항목, 반복 없음 | 수익인식·영업권·매출채권 KAM 등재 |
| **후속사건** | 소규모 또는 없음 | 대규모 소송, 인수, 자금조달 기재 |
| **MD&A vs 실제** | 설명과 숫자 일치 | 낙관적 MD&A + 실적 하락 불일치 |
| **계속기업 불확실성** | 없음 | 등재 → 즉시 유동성 점검 |
| **Fair Value Level 3** | 소규모 | 전체 금융자산의 30%+ |

---

### ⑤ DART 실전 박스

```
투자자의 사업보고서 읽기 순서 (권장):

1. 감사보고서 → KAM 확인 (감사인의 위험 지도)
2. 주석 1번 → 중요 회계정책 (수익인식, 자산화 기준)
3. 주석 → 우발채무 및 약정사항 (숨겨진 부채)
4. 주석 → 특수관계자 거래 (이해충돌 탐지)
5. 주석 → 차입금 만기구조 (리파이낸싱 리스크)
6. 주석 → 연금 적립 현황 (숨겨진 의무)
7. 재무제표 본문 (주석을 이해한 후에 읽으면 달리 보인다)
```

### 📊 내재가치 최종 체크리스트 (이 책을 마치며)

```
이 기업의 사업보고서를 열고 다음을 확인했는가:

□ KAM 항목을 읽고 집중 분석 대상 주석을 식별했는가
□ 우발채무 최대 노출액을 기업가치에 반영했는가
□ 차입약정 조건과 현재 지표의 여유를 계산했는가
□ 연금 적립 부족액을 순부채에 더했는가
□ 특수관계자 매출 비중을 계산해 외부 경쟁력을 분리했는가
□ CFO/순이익 비율, FCF/매출, ROIC를 3년치 계산했는가
□ 역DCF로 시장의 성장 기대치를 역산했는가
□ 적정 PBR과 현재 PBR을 ROE/WACC로 비교했는가

이 체크리스트를 채울 수 있다면 —
이 책의 목적은 달성된 것이다.
```
