# 1.2 재무제표 연결 구조: B/S → I/S → C/F → SCE의 데이터 파이프라인

*— 네 개의 보고서는 하나의 연결된 시스템이다*

---

### ① 시그널 도입부 — 이 개념이 잘못 이해되면 어떤 투자 실수를 만드는가

2022년 가을, 한 투자자가 코스피 중형 제조기업 C사를 발견했다. 3년 평균 순이익 1,100억, PBR 0.75배. DART에서 손익계산서를 열고, 재무상태표를 열고, 닫았다. "저평가"라고 판단했다.

그는 세 번째 보고서를 열지 않았다.

2년 후, 그는 -31%의 손실을 보고 있었다. 누적 OCI 마이너스가 자본의 22%를 잠식했고, 해외 자회사 매각 시 환산손실 1,200억이 손익계산서로 쏟아져 나왔다. "어디서도 경고를 못 봤다"고 그는 말했다. 그러나 경고는 처음부터 자본변동표에 있었다.

많은 투자자가 재무제표를 네 개의 별도 문서로 이해한다.

손익계산서는 이익을 보는 곳. 재무상태표는 재산을 보는 곳. 현금흐름표는 현금을 보는 곳. 자본변동표는 잘 모르겠으니 일단 패스.

이 방식으로 분석하면 결정적인 함정을 놓친다.

어떤 기업의 당기순이익이 1,200억이었다. 배당도 꾸준했다. 안정적으로 보였다. 그런데 그해 자본 총계는 오히려 **340억 감소**했다. 이익이 1,200억이나 났는데 자본이 줄어들었다. 어떻게 된 일인가.

답은 자본변동표에 있었다. 해외 자회사 환산 손실 890억, 확정급여형 연금 보험수리적 손실 650억이 OCI(기타포괄손익)로 자본을 직접 잠식하고 있었다. 이 손실들은 손익계산서를 통과하지 않는다. 자본변동표를 열지 않으면 보이지 않는다.

재무제표는 독립된 보고서가 아니다. **하나의 연결된 시스템**이다.

순이익은 이익잉여금을 통해 재무상태표의 자본으로 흘러들어간다. OCI 항목은 자본을 직접 건드린다. 현금흐름표는 재무상태표의 현금 변동을 설명한다. 이 연결고리를 이해하면 어떤 기업의 재무제표를 열어도 전체 구조가 한눈에 들어온다.

그 구조가 어떻게 작동하는지 직접 들여다보자.

### ② 알고리즘 설명 — 재무제표의 연결 구조

#### 네 개의 보고서, 하나의 흐름

재무제표는 하나의 사이클로 연결된다.

```
① 손익계산서(I/S)
   한 해 동안 얼마를 벌었나
        ↓ 당기순이익
② 자본변동표(SCE)
   이익잉여금 반영 + OCI 항목 직접 반영
        ↓ 기말 자본
③ 재무상태표(B/S)
   기말 시점의 자산 · 부채 · 자본 스냅샷
        ↑ 현금 잔액
④ 현금흐름표(C/F)
   기초 현금 → 기말 현금으로의 이동 경로 설명
```

이 네 개가 맞물려 돌아간다. 하나를 바꾸면 나머지에 파급된다. 그래서 이 연결을 **Articulation(재무제표 연결)** 이라 부른다.

#### 각 보고서의 역할과 핵심 질문

**재무상태표(B/S) — 특정 시점의 스냅샷**

```
자산 = 부채 + 자본
(기업이 보유한 것) = (빌린 것) + (자기 것)
```

재무상태표는 영화의 한 장면이다. 특정 날짜의 기업 재산 상태를 찍어놓은 사진. 흐름이 아니라 **잔액**이다.

핵심 질문: 이 기업은 지금 무엇을 가지고 있고, 누구에게 무엇을 빚지고 있는가.

**손익계산서(I/S) — 일정 기간의 성과**

```
매출 - 비용 = 영업이익
영업이익 ± 영업외손익 - 세금 = 당기순이익
```

손익계산서는 영화의 전체 스토리다. 1년간 기업이 얼마를 벌고 썼는지의 흐름. 발생주의 기준으로 작성된다(1.1에서 다룬 내용).

핵심 질문: 이 기업은 돈을 잘 벌고 있는가.

**현금흐름표(C/F) — 현금의 실제 이동**

```
영업활동(CFO) + 투자활동(CFI) + 재무활동(CFF) = 현금 순증감
기초현금 + 현금 순증감 = 기말현금
```

현금흐름표는 통장 내역서다. 발생주의 이익이 아니라 실제로 들어오고 나간 현금의 기록.

핵심 질문: 이 기업은 실제로 현금을 만들고 있는가.

**자본변동표(SCE) — 자본의 변화 경로**

```
기초자본 + 당기순이익 + OCI ± 자본거래 = 기말자본
```

자본변동표는 재무상태표의 자본이 어떻게 변했는지를 설명하는 다리다. 손익계산서를 통과하지 않고 자본을 직접 건드리는 항목들(OCI)이 여기서 드러난다.

핵심 질문: 손익계산서에 잡히지 않은 자본 변동이 있는가.

#### OCI — 손익계산서를 우회하는 손익

OCI(기타포괄손익)는 당기 손익에 포함되지 않지만 자본을 직접 변동시키는 항목들이다.

| OCI 항목 | 발생 원인 | 재무제표 영향 |
|---|---|---|
| 해외사업 환산차이 | 해외 자회사 환율 변동 | 자본 직접 증감 |
| 확정급여형 연금 보험수리적손익 | 연금 가정 변화 | 자본 직접 증감 |
| 매도가능금융자산 평가손익 | 보유 주식 공정가치 변동 | 자본 직접 증감 |
| 현금흐름 헤지 손익 | 파생상품 헤지 효과 | 자본 직접 증감 |

이 항목들은 손익계산서에 나타나지 않는다. 그러나 기업의 실질 자산가치를 변동시킨다. 누적 OCI 마이너스가 커지면, 언제가 됐든 손익계산서로 터져 나올 잠재 손실이 쌓이고 있다는 뜻이다.

#### 연결의 실제: 숫자로 확인하기

한 기업의 간단한 재무제표를 통해 연결 구조를 확인해보자.

```
[손익계산서]
매출        1,000억
영업이익      100억
당기순이익     75억   ──────────────────────┐
                                           ↓
[자본변동표]                          [이익잉여금에 가산]
기초이익잉여금  500억
+ 당기순이익    75억
- 배당금       -20억
OCI(환산손실)  -30억  ← 손익계산서 통과 안 함
기말이익잉여금  525억
기말자본       890억  ──────────────────────┐
                                           ↓
[재무상태표(기말)]                    [자본에 반영]
총자산       2,100억
총부채       1,210억
총자본         890억  ✓ 일치

[현금흐름표]
기초현금       150억
CFO           +95억
CFI           -60억
CFF           -25억
기말현금       160억  ──────────────────────→ [재무상태표 현금과 일치]
```

이 숫자들이 서로 맞물려 돌아간다는 것을 확인하라. 어느 하나라도 불일치가 생기면 오류가 있다는 뜻이다. 그리고 투자자로서 이 연결 구조를 알면 — 자본 변동이 순이익과 다를 때 즉시 OCI를 의심하게 된다.

### ③ 📋 케이스 스터디 — "OCI에서 터진 폭탄"

**코스피 대형 제조기업 C사 | 3년간 재무제표 흐름 분석**

---

**그날의 분석**

C사는 코스피 상장 중형 제조기업이었다. 3년간 꾸준히 순이익 흑자를 냈고, 배당도 매년 빠지지 않았다. 부채비율은 85% 수준으로 안정적이었다.

한 투자자가 C사를 분석하면서 손익계산서와 재무상태표만 봤다. 3년 평균 순이익 1,100억, PBR 0.75배. 저평가처럼 보였다.

그런데 자본변동표를 열었을 때 의아한 숫자가 나왔다.

---

**숫자 — 3년간의 자본 변동**

| 항목 | 1년차 | 2년차 | 3년차 | 누계 |
|---|---|---|---|---|
| 당기순이익 | +1,050억 | +1,120억 | +1,230억 | +3,400억 |
| 배당금 지급 | -280억 | -310억 | -350억 | -940억 |
| **OCI: 해외사업 환산손실** | -420억 | -530억 | -610억 | **-1,560억** |
| **OCI: 연금 보험수리적손실** | -180억 | -220억 | -250억 | **-650억** |
| **자본 총계 순변동** | **+170억** | **+60억** | **+20억** | **+250억** |

3년간 순이익 합계 3,400억에서 배당 940억을 빼면 자본이 2,460억 늘어야 한다. 그런데 실제 자본 증가는 250억이었다. 사라진 2,210억이 바로 OCI였다.

해외 자회사 환산손실과 확정급여형 연금 보험수리적 손실이 해마다 쌓이고 있었다. 이 손실들은 손익계산서 어디에도 나타나지 않았다.

---

**해부 — 왜 보이지 않았는가**

두 OCI 항목의 성격은 다르다.

**해외사업 환산손실**은 원화 강세 기조 속에 해외 자회사 실적이 원화로 환산되면서 생긴 손실이다. 자회사의 실제 영업 성과와 무관하게 환율 변동 하나로 수백억이 사라진다. 이 손실은 자본을 잠식하지만 손익계산서에는 나타나지 않는다.

**연금 보험수리적 손실**은 확정급여형 퇴직연금의 채무 추정 변화에서 온다. 할인율, 임금 상승률, 기대여명 등의 가정이 변하면 PBO(예측급여채무)가 커지고, 그 차이가 OCI로 처리된다. 지금 당장 현금이 나가지 않지만, 미래에 나가야 할 돈이 늘어난 것이다.

---

**함정 — 왜 대부분이 놓쳤는가**

손익계산서의 순이익 1,100억은 진짜 숫자다. 틀리지 않았다.

그러나 그 숫자만 보면 자본이 빠르게 잠식되고 있다는 사실을 알 수 없다. PBR 0.75배가 저평가처럼 보이는 이유는 자본이 줄어들지 않았기 때문이다. 그런데 OCI로 인해 자본은 이미 잠식되고 있었다.

3년 후 시점에서 이 기업의 수정된 PBR을 계산하면 달라진다. OCI로 사라진 2,210억을 반영하면 실질 자본은 훨씬 작다. '저평가'처럼 보이던 숫자의 근거가 흔들린다.

---

**결말**

4년차, C사는 적자가 지속되던 동남아 자회사를 매각했다. 그 순간 누적 환산손실 1,200억이 OCI에서 손익계산서로 전환됐다. 해당 분기 당기순손실 980억. 자본변동표에서 4년간 조용히 쌓여오던 잠재 손실이 한꺼번에 현실이 된 것이다.

주가는 자회사 매각 공시 당일 -14%, 이후 3개월간 추가로 -17% 하락했다. 3년간 PBR 0.75배가 "저평가"라 믿고 보유했던 투자자들은 누적 -31%의 손실을 안았다.

---

**교훈**

자본변동표는 재무제표 중 가장 자주 무시되는 보고서다. 그리고 가장 많은 진실이 숨겨져 있는 보고서다.

> B/S의 자본이 I/S의 순이익만큼 늘지 않을 때,
> 반드시 자본변동표를 열어 OCI 항목을 추적하라.
> OCI 누적 마이너스는 언젠가 손익계산서로 전환될 잠재 손실이다.
> C사의 투자자가 놓친 것은 숫자가 아니었다. 네 번째 보고서를 열지 않은 것이었다.

### ④ 투자자 시그널 대조표

| 구분 | 긍정(+) 시그널 | 부정(-) 시그널 |
|---|---|---|
| **자본 변동 일치** | 자본 증가 ≈ 순이익 - 배당 | 자본 증가 << 순이익 - 배당 |
| **누적 OCI** | 플러스 또는 0에 근접 | 마이너스 누적, 자본 대비 10% 초과 |
| **OCI 항목** | 소규모, 일시적 | 환산손실·연금손실이 매년 반복 |
| **재무제표 연결** | 네 보고서 숫자가 정합적 | 불일치 → 오류 또는 복잡한 자본 거래 |
| **주석 공시** | OCI 원인과 규모 명확히 기재 | 주석이 모호하거나 OCI 설명 부족 |

### ⑤ DART 실전 박스

**K-IFRS vs US GAAP 차이**

K-IFRS는 포괄손익계산서에 당기순이익과 OCI를 통합 표시하거나 분리 표시할 수 있다.
US GAAP은 OCI를 주로 자본변동표(SCE)에 표시한다.
형식은 다르지만 내용은 같다. 한국 기업 분석 시 포괄손익계산서 하단의 OCI 항목을 반드시 확인하라.

**DART에서 이렇게 확인한다**

```
Step 1. 사업보고서 → [III. 재무에 관한 사항] → [재무제표]
        읽는 순서: 재무상태표 → 포괄손익계산서 → 현금흐름표 → 자본변동표 → 주석

Step 2. 포괄손익계산서 하단 → OCI 항목 확인
        해외사업환산차이, 보험수리적손익, 기타포괄손익 소계 규모 확인

Step 3. 자본변동표 → 기타포괄손익누계액 열 추적
        3개년 누적 OCI 방향(플러스/마이너스)과 규모 확인

Step 4. 검증: 자본 순변동 = 순이익 - 배당 + OCI ± 자본거래
        이 등식이 성립하지 않으면 자본 직접 거래 추가 확인

Step 5. 주석 → [기타포괄손익 누계액] 항목
        OCI 각 구성 요소의 성격과 향후 손익 전환 가능성 파악
```

### 📊 내재가치 체크리스트

**Step 1 — 자본 변동 검증**
```
자본 순증가 계산:
  예상 자본 증가 = 당기순이익 - 배당금
  실제 자본 증가 = 기말자본 - 기초자본

  괴리 = 예상 - 실제
  → 괴리가 크면 OCI 항목 규모 확인
```

**Step 2 — 누적 OCI 부담 계산**
```
누적 OCI 마이너스 / 기말자본 = OCI 자본 잠식 비율

판정 기준:
  5% 미만  → 관리 가능 수준
  5~15%   → 주의 구간, 원인 파악 필요
  15% 초과 → OCI 리스크가 자본가치에 중대한 영향
```

**Step 3 — 종합 판정**
```
OCI 항목이 매년 마이너스이고 누적 잠식이 10% 초과
→ PBR 기반 밸류에이션 시 OCI 잠재 손실을 자본에서 차감하여 재계산
→ 조정 PBR = 시가총액 / (자본 - 누적 OCI 마이너스)
```

대부분의 투자자는 손익계산서 하나만 본다. 일부는 재무상태표까지 연다.
그러나 네 번째 보고서 — 자본변동표 — 를 여는 사람은 거의 없다.
지금 DART에서 관심 종목의 자본변동표를 열어라. 3년간 각 열의 방향을 추적하라.
네 번째 보고서를 읽는 순간, 세 개만 읽던 시절로 돌아갈 수 없다.
