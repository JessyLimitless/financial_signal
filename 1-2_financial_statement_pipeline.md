## 1.2 재무제표 연결 구조 — 네 번째 보고서를 열지 않은 대가

연말 세금 정산을 하다가 포트폴리오를 점검했다.

코스피 중형 제조기업 C사의 사업보고서를 DART에서 다시 열었다. 3개월째 추적하던 종목이었다. 커뮤니티에서 '저평가 가치주'로 자주 올라오는 이름이었다.

PDF가 열렸다. 손익계산서부터 훑었다.

3년 평균 순이익 1,100억.
배당도 매년 빠지지 않았다.
부채비율 85%.

재무상태표를 열었다. PBR(주가순자산비율, 주가가 장부가치의 몇 배인지) 0.75배.

'이 정도면 저평가 아닌가.'

나는 재무상태표를 닫았다.

세 번째 보고서, 현금흐름표도 한 번 훑었다. 괜찮았다. 닫았다.

네 번째 보고서는 열지 않았다. 자본변동표. 그게 뭔데. 뭐가 거기 있는데.

매수 버튼을 눌렀다. 확신이 있었다. PBR 0.75배에 배당까지 나오는 흑자 기업이니까.

2년 후, 나는 -31%의 손실을 보고 있었다.

누적 기타포괄손익(OCI, 손익계산서를 거치지 않고 자본을 직접 변동시키는 항목) 마이너스가 자본의 22%를 잠식했고, 해외 자회사 매각 시 환산손실 1,200억이 손익계산서로 쏟아져 나왔다.

'어디서도 경고를 못 봤는데.'

그러나 경고는 처음부터 자본변동표에 있었다.

---

### 자본변동표를 열었을 때

다시 2년 전으로 돌아가자. 만약 그날 밤, 네 번째 보고서를 열었다면.

C사의 3년 자본변동표를 펼치면 이런 숫자가 나온다.

| 항목 | 1년차 | 2년차 | 3년차 | 누계 |
|---|---|---|---|---|
| 당기순이익 | +1,050억 | +1,120억 | +1,230억 | +3,400억 |
| 배당금 지급 | -280억 | -310억 | -350억 | -940억 |
| **OCI: 해외사업 환산손실** | -420억 | -530억 | -610억 | **-1,560억** |
| **OCI: 연금 보험수리적손실** | -180억 | -220억 | -250억 | **-650억** |
| **자본 총계 순변동** | **+170억** | **+60억** | **+20억** | **+250억** |

숫자를 한 줄씩 따라가보자.

3년간 순이익 합계 3,400억.
배당 940억을 빼면 자본이 2,460억 늘어야 한다.
그런데 실제 자본 증가는 250억이었다.

사라진 2,210억.

숫자를 두 번 읽었다. '이 돈은 어디로 갔는가.'

답은 OCI였다. 해외사업 환산손실 1,560억. 연금 보험수리적손실 650억. 합계 2,210억이 손익계산서를 거치지 않고 자본을 직접 깎아먹고 있었다.

어떻게 이익이 3,400억이나 났는데 자본이 250억밖에 늘지 않을 수 있는가. 이 질문에 답하려면 재무제표 네 개가 어떻게 연결되어 있는지를 이해해야 한다.

---

### 네 개의 보고서는 하나의 시스템이다

많은 투자자가 재무제표를 네 개의 별도 문서로 이해한다.

손익계산서는 이익을 보는 곳. 재무상태표는 재산을 보는 곳. 현금흐름표는 현금을 보는 곳. 자본변동표는 잘 모르겠으니 일단 패스.

나도 그랬다. 세 개면 충분하다고 생각했다.

이 방식으로 분석하면 결정적인 함정을 놓친다. 재무제표는 독립된 보고서가 아니다. **하나의 연결된 시스템**이다. 이 연결을 **Articulation(재무제표 연결)** 이라 부른다.

```
손익계산서(I/S, Income Statement)
   한 해 동안 얼마를 벌었나
        ↓ 당기순이익
자본변동표(SCE, Statement of Changes in Equity)
   이익잉여금 반영 + OCI 항목 직접 반영
        ↓ 기말 자본
재무상태표(B/S, Balance Sheet)
   기말 시점의 자산 · 부채 · 자본 스냅샷
        ↑ 현금 잔액
현금흐름표(C/F, Cash Flow Statement)
   기초 현금 → 기말 현금으로의 이동 경로 설명
```

이 네 개가 맞물려 돌아간다. 하나를 바꾸면 나머지에 파급된다.

각각의 보고서가 묻는 질문은 다르다.

**재무상태표(B/S)** 는 특정 시점의 스냅샷이다. 영화의 한 장면. 흐름이 아니라 **잔액**이다.

```
자산 = 부채 + 자본
(기업이 보유한 것) = (빌린 것) + (자기 것)
```

핵심 질문: 이 기업은 지금 무엇을 가지고 있고, 누구에게 무엇을 빚지고 있는가.

**손익계산서(I/S)** 는 일정 기간의 성과다. 영화의 전체 스토리. 1년간 기업이 얼마를 벌고 썼는지의 흐름. 발생주의 기준으로 작성된다(1.1에서 다룬 내용).

```
매출 - 비용 = 영업이익
영업이익 ± 영업외손익 - 세금 = 당기순이익
```

핵심 질문: 이 기업은 돈을 잘 벌고 있는가.

**현금흐름표(C/F)** 는 통장 내역서다. 발생주의 이익이 아니라 실제로 들어오고 나간 현금의 기록.

```
영업활동(CFO) + 투자활동(CFI) + 재무활동(CFF) = 현금 순증감
기초현금 + 현금 순증감 = 기말현금
```

핵심 질문: 이 기업은 실제로 현금을 만들고 있는가.

**자본변동표(SCE)** 는 재무상태표의 자본이 어떻게 변했는지를 설명하는 다리다. 손익계산서를 통과하지 않고 자본을 직접 건드리는 항목들(OCI)이 여기서 드러난다.

```
기초자본 + 당기순이익 + OCI ± 자본거래 = 기말자본
```

핵심 질문: 손익계산서에 잡히지 않은 자본 변동이 있는가.

C사에서 내가 놓친 것이 바로 이 마지막 질문이었다.

---

### OCI — 손익계산서를 우회하는 손익

C사의 자본을 잠식한 것은 OCI(기타포괄손익)였다. 당기 손익에 포함되지 않지만 자본을 직접 변동시키는 항목들이다.

| OCI 항목 | 발생 원인 | 재무제표 영향 |
|---|---|---|
| 해외사업 환산차이 | 해외 자회사 환율 변동 | 자본 직접 증감 |
| 확정급여형 연금 보험수리적손익 | 연금 가정 변화 | 자본 직접 증감 |
| FVOCI 선택 지분상품 평가손익 | 보유 주식 공정가치 변동 | 자본 직접 증감 (처분 시에도 NI 전환 불가 — 영구 OCI) |
| 현금흐름 헤지 손익 | 파생상품 헤지 효과 | 자본 직접 증감 |

이 항목들은 손익계산서에 나타나지 않는다. 그러나 기업의 실질 자산가치를 변동시킨다. 누적 OCI 마이너스가 커지면, 언제가 됐든 손익계산서로 터져 나올 잠재 손실이 쌓이고 있다는 뜻이다.

C사에서 일어난 일이 정확히 그것이었다.

---

### 보이지 않던 손실의 정체

C사를 잠식한 두 OCI 항목의 성격은 달랐다.

**해외사업 환산손실**. 원화 강세 기조 속에 해외 자회사 실적이 원화로 환산되면서 생긴 손실이었다. 자회사의 실제 영업 성과와 무관하게 환율 변동 하나로 수백억이 사라진다. 이 손실은 자본을 잠식하지만 손익계산서에는 나타나지 않는다.

**연금 보험수리적 손실**. 확정급여형 퇴직연금의 채무 추정 변화에서 왔다. 할인율, 임금 상승률, 기대여명 등의 가정이 변하면 PBO(예측급여채무)가 커지고, 그 차이가 OCI로 처리된다. 지금 당장 현금이 나가지 않지만, 미래에 나가야 할 돈이 늘어난 것이다.

손익계산서의 순이익 1,100억은 진짜 숫자다. 틀리지 않았다. 그러나 그 숫자만 보면 자본이 빠르게 잠식되고 있다는 사실을 알 수 없다.

PBR 0.75배가 저평가처럼 보이는 이유는 자본이 줄어들지 않았다고 착각했기 때문이다. OCI로 인해 자본은 이미 잠식되고 있었다. 3년 후 시점에서 OCI로 사라진 2,210억을 반영하면 실질 자본은 훨씬 작다. '저평가'처럼 보이던 숫자의 근거가 흔들린다.

이 연결 구조를 숫자로 직접 확인하면 더 명확하다.

```
[손익계산서]
매출        1,000억
영업이익      100억
당기순이익     75억   ──────────────────────┐
                                           ↓
[자본변동표]                          [이익잉여금에 가산]
기초이익잉여금  500억
+ 당기순이익    75억
- 배당금       -20억
OCI(환산손실)  -30억  ← 손익계산서 통과 안 함
기말이익잉여금  525억
기말자본       890억  ──────────────────────┐
                                           ↓
[재무상태표(기말)]                    [자본에 반영]
총자산       2,100억
총부채       1,210억
총자본         890억  ✓ 일치

[현금흐름표]
기초현금       150억
CFO           +95억
CFI           -60억
CFF           -25억
기말현금       160억  ──────────────────────→ [재무상태표 현금과 일치]
```

이 숫자들이 서로 맞물려 돌아간다는 것을 확인하라. 어느 하나라도 불일치가 생기면 오류가 있다는 뜻이다. 그리고 투자자로서 이 연결 구조를 알면 — 자본 변동이 순이익과 다를 때 즉시 OCI를 의심하게 된다.

### 조용히 쌓이던 폭탄이 터지다

4년차의 일이다.

새벽에 DART 공시 알림이 왔다. C사가 적자가 지속되던 동남아 자회사를 매각했다. 뉴스만 보면 '비효율 자산 정리'라는 긍정적 이벤트였다.

그런데 그 순간, 누적 환산손실 1,200억이 OCI에서 손익계산서로 전환됐다.

해당 분기 당기순손실 980억.

자본변동표에서 4년간 조용히 쌓여오던 잠재 손실이 한꺼번에 현실이 된 것이다. 비유하자면 — 집 안에 물이 한 방울씩 새고 있었는데, 어느 날 벽을 허무니 곰팡이가 온 벽면을 덮고 있었다. 물이 새기 시작한 날부터 이미 집은 망가지고 있었다.

주가는 자회사 매각 공시 당일 -14%. 이후 3개월간 추가로 -17% 하락했다. 3년간 PBR 0.75배가 "저평가"라 믿고 보유했던 사람들은 누적 -31%의 손실을 안았다.

나는 그중 한 명이었다. 네 번째 보고서를 열지 않은 대가였다.

> B/S의 자본이 I/S의 순이익만큼 늘지 않을 때,
> 반드시 자본변동표를 열어 OCI 항목을 추적하라.
> OCI 누적 마이너스는 언젠가 손익계산서로 전환될 잠재 손실이다.
> C사의 투자자가 놓친 것은 숫자가 아니었다. 네 번째 보고서를 열지 않은 것이었다.

### 시그널 대조표

C사의 교훈을 일반화하면 다음과 같은 시그널 체계가 나온다.

| 구분 | 긍정(+) 시그널 | 부정(-) 시그널 |
|---|---|---|
| **자본 변동 일치** | 자본 증가 ≈ 순이익 - 배당 | 자본 증가 << 순이익 - 배당 |
| **누적 OCI** | 플러스 또는 0에 근접 | 마이너스 누적, 자본 대비 10% 초과 |
| **OCI 항목** | 소규모, 일시적 | 환산손실·연금손실이 매년 반복 |
| **재무제표 연결** | 네 보고서 숫자가 정합적 | 불일치 → 오류 또는 복잡한 자본 거래 |
| **주석 공시** | OCI 원인과 규모 명확히 기재 | 주석이 모호하거나 OCI 설명 부족 |

**K-IFRS vs US GAAP 차이**

K-IFRS는 포괄손익계산서에 당기순이익과 OCI를 통합 표시하거나 분리 표시할 수 있다.
US GAAP은 OCI를 주로 자본변동표(SCE)에 표시한다.
형식은 다르지만 내용은 같다. 한국 기업 분석 시 포괄손익계산서 하단의 OCI 항목을 반드시 확인하라.

**자본 변동 검증법**

```
자본 순증가 계산:
  예상 자본 증가 = 당기순이익 - 배당금
  실제 자본 증가 = 기말자본 - 기초자본

  괴리 = 예상 - 실제
  → 괴리가 크면 OCI 항목 규모 확인
```

**누적 OCI 부담 계산**

```
누적 OCI 마이너스 / 기말자본 = OCI 자본 잠식 비율

판정 기준:
  5% 미만  → 관리 가능 수준
  5~15%   → 주의 구간, 원인 파악 필요
  15% 초과 → OCI 리스크가 자본가치에 중대한 영향
```

> 참고: KOSPI 200 기업 중 누적 OCI/자본이 15%를 초과하는 기업은 약 8~10%이며, 이들 기업의 PBR은 구조적으로 할인되어 거래되는 경향이 있다.

**종합 판정**

```
OCI 항목이 매년 마이너스이고 누적 잠식이 10% 초과
→ PBR 기반 밸류에이션 시 OCI 잠재 손실을 자본에서 차감하여 재계산
→ 조정 PBR = 시가총액 / (자본 - 누적 OCI 마이너스)
```

### 실전: DART에서 확인하는 법

```
Step 1. 사업보고서 → [III. 재무에 관한 사항] → [재무제표]
        읽는 순서: 재무상태표 → 포괄손익계산서 → 현금흐름표 → 자본변동표 → 주석

Step 2. 포괄손익계산서 하단 → OCI 항목 확인
        해외사업환산차이, 보험수리적손익, 기타포괄손익 소계 규모 확인

Step 3. 자본변동표 → 기타포괄손익누계액 열 추적
        3개년 누적 OCI 방향(플러스/마이너스)과 규모 확인

Step 4. 검증: 자본 순변동 = 순이익 - 배당 + OCI ± 자본거래
        이 등식이 성립하지 않으면 자본 직접 거래 추가 확인

Step 5. 주석 → [기타포괄손익 누계액] 항목
        OCI 각 구성 요소의 성격과 향후 손익 전환 가능성 파악
```
