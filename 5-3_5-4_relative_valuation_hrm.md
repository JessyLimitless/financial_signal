# 5.3 상대가치 평가: PER·PBR·EV/EBITDA의 해석

*— 배수 하나로 싸다/비싸다 말하는 것이 왜 위험한가*

---

### ① 시그널 도입부

PBR 0.4배. 역사적 저점이다. 매수 타이밍처럼 보인다.

그런데 이 기업의 ROE는 3.2%다. WACC는 9%다.

ROE < WACC인 기업은 자본을 투입할수록 손실이 난다. 이런 기업에서 PBR이 1배를 회복하려면 ROE가 WACC를 초과하는 날이 와야 한다. 그 날이 오지 않으면 PBR 0.4배는 저평가가 아니라 적정 가치다.

```
적정 PBR = ROE / (WACC - g)
         = 3.2% / (9% - 1%)
         = 0.4배
```

PBR 0.4배는 싼 게 아니었다. 수익성이 낮다는 사실을 정직하게 반영한 결과였다.

이것이 멀티플을 올바르게 읽는 법이다. 숫자 자체가 아니라 그 숫자를 만든 분자와 분모를 이해해야 한다.

낮은 PBR과 낮은 PER가 왜 함정이 될 수 있는지, 밸류 트랩의 해부가 시작된다.

---

### ② 알고리즘 설명

#### 주요 멀티플의 분자·분모 해부

**PER (Price / EPS)**
```
분자: 주가 (시장의 기대가 반영)
분모: EPS (지난 이익 또는 예상 이익)

함정:
  - 분모가 Non-GAAP 이익이면 → GAAP으로 재계산
  - 일회성 이익으로 EPS 급증한 해 → 정상화 EPS 사용
  - 사이클 기업 → CAPE(경기조정PER) 적용
```

**PBR (Price / Book Value)**
```
분자: 주가
분모: 주당순자산 (장부가치)

적정 PBR = ROE / (WACC - g)
→ ROE가 WACC를 초과할수록 프리미엄 정당화
→ ROE < WACC인 기업의 PBR < 1은 가치함정일 수 있음

영업권, 개발비 등 무형자산 과다 기업 →
유형순자산(Tangible Book Value)로 PBR 재계산 권장
```

**EV/EBITDA**
```
EV = 시가총액 + 순부채 + 소수주주지분
EBITDA = 영업이익 + 감가상각

장점: 자본 구조와 세율 차이를 제거하고 비교
단점: CAPEX 무시 → 자본집약적 업종에서 왜곡

EV/FCF가 EV/EBITDA보다 실질에 가까움
```

**PEG (PER / 성장률)**
```
PEG = PER / 예상 EPS 성장률
1배 이하 → 성장 대비 저평가 신호 (Peter Lynch 기준)

함정: 성장률 추정의 신뢰성이 핵심
```

#### 상대가치의 함정: 모두 비싸면 비교 의미 없다

업종 전체가 버블에 있을 때 상대 비교는 의미 없다. "이 기업이 업종 내에서 가장 싸다"는 것이 절대적 저평가를 뜻하지 않는다.

상대가치는 **절대가치(DCF)와 함께** 사용해야 한다.

---

### ③ 📋 케이스 스터디 — "PBR 0.4배, 가치함정인가 진짜 저평가인가"

**코스피 중형 제조기업 비교 | PBR 분해 분석**

---

| 항목 | P사 | Q사 |
|---|---|---|
| PBR | 0.4배 | 0.4배 |
| ROE | 3.2% | 14.8% |
| WACC 추정 | 9% | 9% |
| 영구성장률 가정 | 1% | 1% |
| **적정 PBR** | **0.4배** | **1.9배** |

P사: PBR 0.4배 = 적정 가치. ROE가 WACC를 크게 하회한다. PBR이 오를 이유가 없다.

Q사: PBR 0.4배 = 저평가. ROE가 WACC를 크게 초과한다. 적정 PBR은 1.9배인데 0.4배에 거래되고 있다.

같은 숫자가 전혀 다른 의미를 갖는다.

---

**왜 Q사는 PBR 0.4배인가**

Q사는 3년간 부채비율이 급등하며 시장의 불신을 샀다. 실적은 좋지만 재무 리스크 우려로 할인된 것이었다. 재무 안정화가 확인되면 PBR이 적정 수준으로 회복될 가능성이 있다.

P사는 수익성 구조 자체가 문제다. 자본을 더 투입해도 WACC를 초과하는 수익을 못 낸다. 배당을 늘리거나 자산을 청산하지 않는 한 PBR은 오르지 않는다.

---

**교훈**

> PBR 숫자보다 ROE와 WACC를 먼저 비교하라.
> ROE > WACC이면서 PBR이 낮으면 → 진짜 저평가 후보.
> ROE < WACC이면서 PBR이 낮으면 → 가치함정. 수익성 개선 촉매가 필요.

---

### ④ 투자자 시그널 대조표

| 구분 | 긍정(+) 시그널 | 부정(-) 시그널 |
|---|---|---|
| **PBR vs ROE/WACC** | ROE > WACC, PBR < 적정 PBR | ROE < WACC, PBR 낮음 (가치함정) |
| **EV/EBITDA vs EV/FCF** | 두 배수 수렴 | EV/EBITDA 낮은데 EV/FCF 높음 |
| **PER 분모** | GAAP 이익, 정상화됨 | Non-GAAP 또는 일회성 이익 |
| **비교군 선정** | 실질 경쟁사, 유사 사업모델 | 업종명만 같은 이질적 기업군 |

---

### ⑤ DART + 멀티플 계산

```
PBR 적정치 계산:
  ROE = 당기순이익 / 자본총계
  g = 유보율 × ROE (지속가능성장률)
  적정 PBR = ROE / (WACC - g)
  현재 PBR vs 적정 PBR 비교

EV 계산:
  EV = 시가총액 + 총차입금 + 리스부채 - 현금
  EV/EBITDA, EV/FCF 계산
```

---

### 📊 내재가치 체크리스트

```
ROE:            (    )%
WACC:           (    )%
g (지속가능):   (    )%
적정 PBR:       (    )배  = ROE / (WACC - g)
현재 PBR:       (    )배
괴리:           (    )배

판정:
  현재 PBR << 적정 PBR + ROE > WACC → 저평가 후보 진입
  현재 PBR ≈ 적정 PBR              → 적정 가격
  현재 PBR < 적정 PBR + ROE < WACC  → 가치함정 주의
```

---

지금 관심 종목의 ROE와 WACC를 계산하라.
적정 PBR을 공식으로 산출하라. 현재 PBR과 비교하라.

그 차이가 보이면, 저평가인지 가치함정인지의 답이 나온다.

# 5.4 종합 판단: 내재가치 시그널 체크리스트 (HRM 연결)

*— 이 책의 모든 분석을 하나의 투자 판단으로 수렴시킨다*

---

### ① 시그널 도입부

이 책의 1장부터 5.3까지를 한 줄로 요약하면 이렇다.

**이 이익은 진짜인가. 이 부채는 다 보이는가. 이 성장은 지속되는가. 그리고 지금 가격에 사는 것이 합리적인가.**

이 네 개의 질문이 투자 판단의 전부다.

HRM(Hierarchical Report Model)은 이 질문들에 대한 답을 수치화한 시스템이다. 수익성, 안전성, 성장성, 효율성, 밸류에이션의 5개 카테고리로 100점을 측정한다. 그리고 정량으로 잡히지 않는 것들(바이오 프리미엄, 경영권 분쟁, 내부자 매수)을 ±10점의 정성 조정으로 보완한다.

이 섹션은 그 시스템의 해설서다.

책의 각 챕터가 HRM의 어느 점수를 설명하는지, 그리고 그 점수가 최종 투자 판단으로 어떻게 이어지는지를 보여준다.

---

### ② HRM 5카테고리 × 책 챕터 매핑

| HRM 카테고리 | 배점 | 이 책의 해설 챕터 | 핵심 지표 |
|---|---|---|---|
| **수익성** | 25점 | 4.1, 4.2, 4.3 | OPM, ROE, CFO/순이익 |
| **안전성** | 25점 | 3.1, 3.2, 3.3 | 부채비율, 이자보상배율, 유동비율 |
| **성장성** | 20점 | 2.5, 4.1 | 매출/영업이익 성장률, ROIC 추세 |
| **효율성** | 15점 | 4.3, 5.1 | FCF/매출, ROIC (Phase 3 OCF 이원화) |
| **밸류에이션** | 15점 | 5.2, 5.3 | PER, PBR, EV/EBITDA |

---

### ③ 정성 조정 ±10점: 어떤 상황에서 적용되는가

```
감점 사유:
  -10점: 불성실공시, 감사의견 비적정
  -5~-8점: 지주회사 디스카운트
  -3~-7점: 경영권 분쟁 진행 중
  -5~-10점: 바이오 프리미엄 (매출 없는 고평가)
  -3~-5점: 이익 조작 경고신호 복수 발견

가점 사유:
  +3~+5점: 대규모 내부자 장내매수
  +2~+5점: 밸류업 프로그램 참여 + 이행
  +3~+5점: 업종 1위 독점적 지위
  +2~+5점: 글로벌 매출 50%+ 확인

감사 크로스체크:
  |정량 - 최종점수| > 15점 → ALERT
  정량 > 70, 최종 < 40 → ALERT (극단적 감점 근거 필요)
  정량 < 30, 최종 > 60 → ALERT (극단적 가점 근거 필요)
```

---

### ④ 📋 케이스 스터디 — "삼성전자 정량 72점의 의미"

**HRM 시뮬레이션: 삼성전자**

---

**HRM 정량 점수 분해**

| 카테고리 | 배점 | 실제 점수 | 근거 |
|---|---|---|---|
| 수익성 | 25 | 18 | OPM 7~9% (반도체 다운사이클), ROE 8% |
| 안전성 | 25 | 22 | 부채비율 낮음, 순현금 포지션 |
| 성장성 | 20 | 9 | 매출 역성장 반영 (다운사이클) |
| 효율성 | 15 | 10 | OPM 기반 대체 (Phase 3 OCF 전환 예정) |
| 밸류에이션 | 15 | 13 | PER/PBR 저평가 구간 |
| **정량 합계** | **100** | **72** | B+ |

---

**LLM 정성 조정**

LLM 주관 점수: 88점 (A)

조정 내용:
- 업종 1위 독점적 지위: +5점
- 글로벌 기업 (해외 매출 80%+): +3점
- 밸류업 프로그램 참여: +2점
- 반도체 사이클 회복 기대: 정성 판단

괴리: 88 - 72 = **16점** → ALERT 기준(15점) 초과

---

**ALERT 해석**

이 경우 ALERT는 "점수가 잘못됐다"는 의미가 아니다. "정성 조정의 근거를 문서화하라"는 의미다.

삼성전자의 경우, 정성 조정 근거가 명확하다.
- 업종 1위 지위: 반도체 사이클이 바닥에서 회복할 때 가장 크게 수혜받는 기업
- 정량 점수가 낮은 이유: 사이클 저점의 일시적 실적 악화
- 기저효과: 다음 상승 사이클에서 성장성 점수 급등 예상

이 근거가 납득 가능하면 정성 조정이 정당하다. 근거 없이 직감으로 16점을 올린 것이라면 의심해야 한다.

---

**교훈**

> HRM 점수는 현재 재무 상태의 스냅샷이다.
> 사이클 기업, 턴어라운드 기업은 정성 조정과 투자논거 문서화가 핵심이다.
> 정량-정성 괴리가 15점을 넘으면 반드시 그 이유를 글로 쓰라.
> 쓸 수 없으면 그 투자 판단은 근거 없는 직감이다.

---

### ⑤ DART 실전 박스

```
Step 1. HRM 정량 점수 산출
        각 카테고리 점수를 이 책의 해당 챕터 체크리스트에서 계산
        수익성(4.1~4.3) + 안전성(3.1~3.3) + 성장성(2.5, 4.1)
        + 효율성(4.3, 5.1) + 밸류에이션(5.2, 5.3)

Step 2. DART → 사업보고서 + 감사보고서
        KAM(핵심감사사항), 감사의견, 불성실공시 이력 확인
        → 정성 감점 사유 존재 여부 판단

Step 3. DART → [최대주주 변경] [임원 주식거래] 공시
        내부자 매수/매도 패턴 확인
        → 정성 가점/감점 사유 판단

Step 4. 정량-정성 괴리 확인
        |정량 점수 - 최종 점수| > 15점 → ALERT
        근거를 문서화할 수 없으면 정성 조정 철회
```
