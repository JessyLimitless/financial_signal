## 5.1 잉여현금흐름(FCF): 기업이 실제로 창출하는 현금

2021년, 코스피 IT서비스 기업 A사와 반도체 장비 기업 B사의 손익계산서를 나란히 놓으면 거의 구분이 안 됐다.

| 항목 | A기업 | B기업 |
|---|---|---|
| 매출 | 5,000억 | 5,000억 |
| 영업이익 | 500억 | 500억 |
| 당기순이익 | 380억 | 370억 |
| EPS | 같은 수준 | 같은 수준 |
| PER | 12배 | 12배 |

시장은 두 기업을 비슷한 가격에 거래했다.

그런데 현금흐름표를 열면 완전히 다른 기업이 나타났다.

| 항목 | A기업 | B기업 |
|---|---|---|
| 영업현금흐름(CFO) | 460억 | 440억 |
| **CAPEX** | **-80억** | **-360억** |
| **FCF** | **380억** | **80억** |
| **FCF/매출** | **7.6%** | **1.6%** |
| **FCF Yield** | **6.3%** | **1.3%** |
| ROIC | 19% | 7% |

같은 이익을 만들기 위해 B기업은 A기업의 4.5배 CAPEX를 쏟아붓는다. A기업은 FCF가 이익과 거의 같다. B기업은 FCF가 이익의 22%에 불과하다. B기업은 벌어들인 돈의 78%를 설비에 재투자해야만 현재 이익을 유지할 수 있다.

이 차이의 핵심이 EBITDA와 FCF의 구분이다.

---

### EBITDA의 허상, FCF의 실체

EBITDA는 기업 인수 시 주로 쓰는 지표다. 이유가 있다. 현금흐름을 과대평가하기 때문이다.

EBITDA = 영업이익 + 감가상각비.

감가상각비를 다시 더하는 이유는 "현금 유출이 없는 비용"이기 때문이다. 맞다. 그런데 감가상각은 이미 지출한 CAPEX의 비용 배분이다. 설비가 닳으면 교체해야 한다. 그 교체 비용(Maintenance CAPEX)은 현금이 나간다. EBITDA는 이 교체 비용을 무시한다.

진짜 현금 창출 능력 = FCF = CFO - CAPEX.

같은 EBITDA 200억이라도, CAPEX가 20억인 기업과 180억인 기업의 FCF는 180억과 20억으로 9배 차이다. EBITDA 멀티플로 같은 가격에 거래된다면, 한쪽은 극도로 저평가고 한쪽은 극도로 고평가다.

A사와 B사가 바로 이 구조였다.

---

### FCF의 두 가지 버전

```
FCFF (기업 잉여현금흐름)
  = EBIT × (1 - 실효세율)  ← NOPAT (세후 순영업이익)
  + 감가상각비 (D&A)        ← 비현금 비용 환입
  - CAPEX                   ← 설비 투자
  - ΔNWC                    ← 순운전자본 증가

FCFE (주주 잉여현금흐름)
  = 순이익
  + D&A
  - CAPEX
  - ΔNWC
  + 순차입 증가 (차입 - 상환)
```

FCFF는 부채와 자기자본 모두에 귀속되는 현금흐름. WACC로 할인해 기업가치 산출.
FCFE는 주주에게만 귀속되는 현금흐름. Cost of Equity로 할인해 주식가치 산출.

CAPEX 전체 중 유지 목적 vs 성장 목적을 분리해야 한다.

```
방법 1: 감가상각비의 70~80%를 Maintenance CAPEX로 추정
방법 2: 회사 공시에서 유지보수 CAPEX 별도 언급 여부 확인
방법 3: 매출 성장이 0인 해의 CAPEX를 Maintenance CAPEX로 추정
```

Maintenance CAPEX가 감가상각비보다 지속적으로 낮으면 자산 노후화를 의심해야 한다. A사의 사업이 오래 건전했던 이유 중 하나가 이것이다. 소프트웨어와 운영 계약 기반 사업으로, 한 번 구축한 시스템이 반복 매출을 만들어냈다. CAPEX가 낮아도 사업이 유지됐다.

B사는 반도체 장비 기업이었다. 수주를 받으면 고가의 장비를 제작해야 했다. 설비를 계속 유지·교체해야 하고, 기술 변화에 맞춰 생산라인을 갱신해야 했다. 버는 돈의 대부분이 사업 유지에 쓰였다. 같은 PER 12배에 거래되지만, A사는 실제로 주주가 가져갈 수 있는 현금이 380억이고 B사는 80억이었다.

---

### FCF Yield — 가격과 가치의 비교

```
FCF Yield = FCF / 시가총액

해석:
  5% 이상   → 가격 대비 현금 창출 매력적
  3~5%     → 보통
  3% 미만   → 성장 기대가 반영된 가격
  마이너스   → 현재 현금 미창출 (성장 투자 중 vs 구조적 적자 구분 필요)
```

A사의 FCF Yield는 6.3%였다. B사는 1.3%였다. 시장이 같은 가격을 매긴 두 기업의 실질 가치가 이렇게 달랐다.

---

### 3년이 증명한 것

| 항목 | A사 | B사 |
|---|---|---|
| 주가 변동 (3년) | **+68%** | **-4%** |
| 누적 FCF (3년) | 1,180억 | 190억 |
| 누적 배당 + 자사주 | 620억 | 80억 |
| 3년차 ROIC | 21% (유지) | 5.8% (하락) |

A사는 3년간 벌어들인 FCF로 배당과 자사주 매입을 실행했다. 주주환원이 실제로 이루어졌다. B사는 벌어들인 현금의 대부분을 설비에 쏟아부었고, 반도체 다운사이클에 진입하면서 ROIC가 WACC 아래로 떨어졌다.

시장은 3년에 걸쳐 이 차이를 가격에 반영했다. PER이나 EPS만 보면 두 기업이 비슷해 보인다. FCF/매출과 ROIC를 함께 보면 전혀 다른 기업이다. 같은 이익이라도 얼마나 적은 자본으로 만드느냐가 기업의 진짜 가치를 결정한다.

---

### 시그널 대조표

| 구분 | 긍정(+) 시그널 | 부정(-) 시그널 |
|---|---|---|
| **FCF/매출** | 10% 이상, 안정 | 5% 미만, 하락 추세 |
| **FCF 전환율** | 0.8 이상 (FCF/순이익) | 0.5 미만 |
| **FCF Yield** | 5% 이상 | 마이너스 지속 |
| **ROIC vs WACC** | 스프레드 플러스, 안정 | 스프레드 마이너스 |
| **FCF 활용** | 배당+자사주 > 차입 | FCF 마이너스인데 배당 지급 |

---

### 실전: FCF 계산법

```
Step 1. NOPAT 계산
        영업이익 × (1 - 유효세율)

Step 2. FCFF 계산
        NOPAT + 감가상각 - CAPEX - ΔNWC

Step 3. FCF Yield
        FCF / 시가총액 (시가총액은 현재 주가 × 발행주식수)

Step 4. 3년 추세
        FCF, FCF/매출, FCF Yield를 3개년 비교
```

### 내재가치 워크시트

```
NOPAT:        (    )억
+ D&A:        (    )억
- CAPEX:      (    )억
- ΔNWC:       (    )억
= FCFF:       (    )억

FCF/매출:     (    )%

판정: ROIC > WACC + FCF Yield > 5% → 내재가치 분석 진행
     ROIC < WACC → 성장 투자 재검토, 밸류에이션 프리미엄 불가
```

# 5.2 DCF 분석: 현재가치로 환산하는 미래 현금

틀렸다.

어떤 사람은 DCF를 정밀 과학처럼 쓴다. 성장률을 소수점까지 추정하고, 할인율을 0.1% 단위로 계산하고, 엑셀에 수백 개의 셀을 채운다. 그리고 "이 기업의 내재가치는 정확히 47,230원"이라는 결론을 낸다.

DCF는 정밀 도구가 아니다. **논리적 사고 도구**다.

Terminal Value가 DCF 총가치의 70~80%를 차지하는 경우가 많다. Terminal Value는 영구 성장률 g와 WACC 단 두 개의 숫자로 결정된다. g를 1% 바꾸면 내재가치가 30~40% 달라진다. 이것이 정밀 과학인가.

그럼에도 DCF를 쓸 수밖에 없는 이유가 있다. 대안(멀티플, 자산가치)은 더 많은 것을 생략한다. TV 의존도를 줄이려면: 명시적 예측 기간을 10~15년으로 늘리거나, TV를 수렴 ROIC 기반으로 계산하거나(TV = NOPAT / WACC), 낙관/비관 시나리오별 TV를 각각 산출하여 내재가치 밴드를 구하라.

DCF의 진짜 가치는 다른 데 있다. **역DCF**다. 현재 주가를 DCF에 역대입해서, 시장이 어떤 성장률을 가정하고 있는지를 역산한다. 그 가정이 현실적인가를 스스로 판단하는 것 — 이것이 DCF를 쓰는 진짜 이유다.

---

### DCF의 구조

```
기업가치(EV) = Σ FCFt / (1+WACC)^t + Terminal Value / (1+WACC)^n

Terminal Value (Gordon Growth Model):
  TV = FCFn × (1+g) / (WACC - g)
  g = 영구 성장률 (보수적으로 2~3% 권장)

주식가치 = 기업가치 - 순부채
주당 내재가치 = 주식가치 / 발행주식수
```

### WACC 계산

```
WACC = E/(E+D) × Ke + D/(E+D) × Kd × (1-t)

Ke (자기자본비용) = Rf + β × (Rm - Rf)
  Rf: 무위험이자율 (10년 국고채 수익률, 약 3.5~4%)
  β: 베타 (시장 위험 민감도)
  Rm-Rf: 주식위험프리미엄 (한국 시장 약 5~6%)

한국 시장 실전: 베타는 FnGuide 또는 KRX에서 3년 주간 수익률 기준을 확인하라. 코스닥 소형주는 소형주 프리미엄 2~3%를 추가 고려하라. 일간 vs 주간, 1년 vs 3년 기간 선택에 따라 베타가 0.8~1.5까지 차이 날 수 있으므로 단일 값보다 범위로 사용하는 것이 현실적이다.

Kd: 차입이자율 (가중평균)
t: 법인세율
```

이 공식들이 실전에서 어떤 판단을 만들어내는지, 코스피 플랫폼 기업 O사의 역DCF가 보여준다.

---

### O사 — 역DCF가 드러낸 시장의 기대

| 항목 | 값 |
|---|---|
| 현재 주가 | 82,000원 |
| 발행주식수 | 1.5억주 |
| 시가총액 | 12.3조 |
| 순부채 | +8,000억 (순부채) |
| **기업가치(EV)** | **13.1조** |
| 현재 FCF (연간) | 3,200억 |
| WACC 추정 | 10% |
| 영구성장률 가정 | 3% |

역DCF 계산:
- 현재 FCF 3,200억이 WACC 10%, 영구성장률 3%로 성장하면 → TV = 3,200 × 1.03 / (0.10 - 0.03) = 47,086억
- 이 TV의 현재가치만으로도 기업가치가 설명됨
- 따라서 시장은 FCF가 현재 3,200억 수준에서 영구적으로 유지된다고 보는 것이 아님

역산 결과: 향후 5년간 FCF 연평균 성장률 **28%**를 가정해야 현재 주가 설명 가능.

O사의 과거 3년 FCF 성장률: 연평균 14%.

시장은 O사가 과거보다 2배 빠른 속도로 FCF를 성장시킬 것이라고 기대하고 있다.

이것이 가능한가. 스스로 질문해야 한다.

- O사의 핵심 시장 성장률은 연 20% 정도다.
- 시장점유율이 더 오를 여지가 있는가.
- 마진 개선의 여지가 있는가.
- 경쟁 강도는.

이 질문들의 답이 "28% 성장이 가능하다"면 현재 주가는 합리적이다. "14%가 더 현실적이다"면 현재 주가는 미래를 과도하게 선반영하고 있다.

---

### 역DCF를 투자 판단으로 전환하는 기준

```
역DCF 절차:
  1. 현재 주가를 주식가치로 사용
  2. 순부채 더해서 기업가치 역산
  3. WACC와 Terminal Value 가정 고정
  4. 이 기업가치를 만족하는 FCF 성장률 역산

해석:
  역산된 성장률이 과거 3년 평균 성장률보다 크면
  → 시장이 과거보다 더 빠른 성장을 기대하고 있음
  → 그 기대가 현실적인가를 스스로 판단
```

역산 성장률 / 자신의 추정 성장률 비율을 계산하라. 0.7배 이하 → 시장이 보수적, 매수 검토. 1.0~1.3배 → 적정 반영, 관망. 1.5배 초과 → 시장이 과도한 성장을 내재, 매수 회피 또는 기존 포지션 축소.

DCF의 결론은 숫자가 아니다. 이 질문들에 대한 자신만의 답이다. DCF로 정확한 내재가치를 계산하려 하지 마라. 역DCF로 시장이 어떤 미래를 가격에 반영하고 있는지를 역산하라. 그 미래가 현실적인지를 스스로 판단하라. 그 판단이 매수 또는 매도의 근거가 된다.

---

### 시그널 대조표

| 구분 | 판단 방향 |
|---|---|
| **역DCF 내재 성장률 < 과거 성장률** | 시장이 비관적. 저평가 가능성 |
| **역DCF 내재 성장률 > 과거 성장률 × 1.5배** | 시장이 낙관적. 기대치 달성 가능한지 검토 |
| **TV > 총가치의 80%** | 영구가치 가정에 결과 좌우. 민감도 분석 필수 |
| **WACC 1% 변화로 가치 30%+ 변동** | 할인율 불확실성 높음. 광범위한 범위로 제시 |

---

### 실전: 역DCF 계산법

```
Step 1. FCF 계산 (4.3, 5.1에서 이미 계산)
Step 2. WACC 추정
        자기자본비용 = 3.5%(국고채) + 베타 × 5.5%
        부채비용 = 평균 차입이자율 × (1 - 세율)
        WACC = 가중평균
Step 3. 역DCF 역산
        현재 EV = 시총 + 순부채
        TV 가정: g = 3%, WACC 사용
        역산: 현재 EV를 만족하는 5년간 FCF 성장률
Step 4. 역산된 성장률 vs 과거 성장률 비교
        판단: 합리적인가 / 과도한가
```
