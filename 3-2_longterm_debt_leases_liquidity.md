# 3.2 장기부채·리스·유동성: 레버리지의 실질과 생존능력

*— 부채비율이 낮아도 내일 망하는 기업이 있다*

---

### ① 시그널 도입부 — 이 개념이 잘못 이해되면 어떤 투자 실수를 만드는가

2020년 3월, 한 항공사가 무너졌다. 부채비율은 180%로 업종 평균 수준이었다. 그런데 전체 차입금의 72%가 1년 내 만기였고, 이자보상배율은 0.8배였다. 코로나가 터지자 차환이 막혔다. 유동비율 67%. 3개월 만에 법정관리를 신청했다. 부채비율이 같은 경쟁사는 살아남았다. 차입금의 80%가 3년 이상 장기였기 때문이다.

기업이 망하는 것은 부채가 많아서가 아니다. **만기에 갚을 현금이 없어서**다.

두 기업이 있다. 둘 다 부채비율 120%.

A기업의 차입금은 대부분 5년 이상 장기이고, 이자보상배율은 8배다. 유동비율은 180%. 여유 있다.

B기업의 차입금 중 65%가 내년에 만기가 돌아온다. 이자보상배율은 1.4배. 유동비율은 88%.

같은 부채비율 120%지만 A기업과 B기업의 위험은 전혀 다르다.

레버리지 분석에서 중요한 것은 두 가지 축이다.

첫째, **레버리지의 규모** — 부채가 얼마나 되는가. 부채비율이 이것을 말한다.

둘째, **레버리지의 질** — 그 부채를 감당할 수 있는가. 이자보상배율, 유동비율, 만기 구조가 이것을 말한다.

부채비율만 보고 안전을 판단하는 것은 첫 번째 축만 보는 것이다. 두 번째 축 없이는 기업이 실제로 채무를 이행할 능력이 있는지 알 수 없다.

그리고 여기에 하나가 더 있다. **리스 부채**. 재무상태표에 보이는 차입금 외에 운용리스 의무까지 포함해야 진짜 레버리지가 보인다.

그 구조를 실전에서 어떻게 분해하는지 들여다보자.

### ② 알고리즘 설명 — 레버리지의 실질을 측정하는 방법

#### 유효이자율법: 채권의 장부가액이 만들어지는 원리

기업이 채권(사채)을 발행할 때 시장 이자율과 표면 이자율이 다르면 할인 또는 할증 발행이 일어난다.

```
액면가 100억, 표면이자율 3%, 시장이자율 5%인 경우:
  발행가(현재가치) = 3억/1.05 + 3억/1.05² + ... + 103억/1.05³
                  = 94.5억 (할인 발행)
  사채할인발행차금 = 5.5억

매 기간 이자비용 계산 (유효이자율법):
  이자비용 = 장부가액 × 시장이자율(5%)
  현금 지급 = 액면가 × 표면이자율(3%)
  할인차금 상각 = 이자비용 - 현금지급 → 장부가액 증가
```

투자자에게 중요한 것은 이것이다. 할인 발행된 채권의 장부가액은 만기에 가까워질수록 액면가로 증가한다. 이자비용은 현금 지급보다 크다. 손익계산서의 이자비용이 실제 현금 지출 이자와 다를 수 있다.

이것이 FCF 계산에 영향을 준다. 할인 발행 사채가 있는 기업에서 비현금 이자비용(할인차금 상각)이 CFO에 가산 조정되므로, 이를 모르면 CFO가 과대평가된 것으로 착각할 수 있다.

#### 리스 회계: 부외항목이 된 부채를 찾아라

ASC 842 / K-IFRS 16 이전에는 운용리스의 미래 지급 의무가 재무상태표에 올라오지 않았다. 임차료를 그냥 비용으로 처리했다. 리스 의무는 부외항목이었다.

지금은 다르다. 12개월을 초과하는 리스는 ROU 자산(사용권자산)과 리스부채로 재무상태표에 올라온다.

```
리스 개시 시:
  ROU 자산 ↑   /  리스 부채 ↑ (동일 금액)

매 기간:
  리스 이자비용 = 리스부채 × 내재이자율
  감가상각비 = ROU 자산 / 리스기간

금융리스 vs 운용리스 (리이용자 기준):
  금융리스: 이자비용 + 감가상각비로 분리 → 초기 비용 많음
  운용리스: 단순 리스료로 처리 → 기간 균등 비용
```

핵심은 이것이다. 운용리스로 분류된 임차 계약도 이제 대부분 재무상태표에 올라온다. 그런데 **단기리스(12개월 이하)** 와 **소액자산 리스**는 면제 선택이 가능하다. 기업이 이 면제를 광범위하게 활용하면 리스 부채 규모가 실질보다 낮게 표시된다.

#### 조정순부채: 진짜 레버리지 계산법

재무상태표에 보이는 차입금만으로 순부채를 계산하면 부족하다.

```
조정순부채 계산:

재무상태표 차입금 합계:             (    )억
+ 리스부채 (재무상태표):            (    )억
+ 단기리스 미래 지급액 (주석):      (    )억  ← 면제 활용분
+ 연금 적립 부족분 (주석):          (    )억  ← 3.3에서 상세
- 현금 및 현금성자산:              (    )억
──────────────────────────────────
조정순부채:                        (    )억
```

#### 이자보상배율과 유동비율: 생존 능력의 두 지표

**이자보상배율(Interest Coverage Ratio)**은 영업이익으로 이자를 몇 번 낼 수 있는지를 측정한다.

```
이자보상배율 = EBIT / 이자비용

판정 기준:
  3배 이상  → 안정적
  1.5~3배  → 주의 구간
  1.5배 미만 → 이자 납부 능력 한계선 근접
  1배 미만  → 영업이익만으로 이자도 못 냄
```

**유동비율(Current Ratio)**은 1년 내 만기 부채를 1년 내 현금화 가능한 자산으로 갚을 수 있는지를 측정한다.

```
유동비율 = 유동자산 / 유동부채

판정 기준:
  150% 이상 → 안정적
  100~150% → 보통
  100% 미만 → 단기 지급 능력 부족
```

그런데 유동비율에는 함정이 있다. 재고자산이 당장 현금화되기 어려운 기업이라면 재고를 제외한 당좌비율이 더 의미 있다.

```
당좌비율 = (유동자산 - 재고자산) / 유동부채
```

### ③ 📋 케이스 스터디 — "운용리스를 자본화하면 전혀 다른 기업"

**코스피 대형 유통기업 K사 | 리스 자본화 후 실질 레버리지 분석**

---

**그날의 분석**

K사는 전국에 300개 이상의 매장을 운영하는 코스피 대형 유통기업이었다. 매장은 모두 임차였다.

재무상태표를 열면 부채비율 92%가 나온다. 우량하다. 안정적이다.

그런데 리스 주석을 열었을 때 다른 숫자가 나왔다.

---

**숫자 — 리스 의무의 실체**

| 항목 | 재무상태표 기준 | 리스 조정 후 |
|---|---|---|
| 총차입금 | 4,200억 | 4,200억 |
| **리스부채 (재무상태표 반영)** | — | **2,100억** |
| **단기리스 미래 지급액 (주석)** | — | **4,700억** |
| 현금 및 현금성자산 | -1,800억 | -1,800억 |
| **조정순부채** | **2,400억** | **9,200억** |
| 자본 총계 | 4,560억 | 4,560억 |
| **부채비율** | **92%** | **조정 후 201%** |
| **Debt/EBITDA** | **1.8배** | **조정 후 5.2배** |
| 이자보상배율 | 8.2배 | 2.1배 |

단기리스 면제를 활용한 미래 지급액 4,700억이 리스부채에 잡히지 않고 있었다.

K사는 임차 매장의 상당수를 1년 단위로 계약을 갱신하는 방식을 썼다. K-IFRS 16에서 12개월 이하 리스는 단기리스로 분류해 부채 인식을 면제할 수 있다. K사가 이 면제를 광범위하게 활용하면서 실질적인 임차 의무 중 상당 부분이 재무상태표 밖에 있었다.

탐지법: 주석에서 '단기리스 비용'과 '리스부채 잔액'을 비교하라. 단기리스 비용 / (리스부채 + 단기리스 비용 현재가치) 비율이 50%를 넘으면, 기업이 단기 계약 갱신 방식으로 부채 인식을 회피하고 있을 가능성이 높다.

---

**해부 — 임차 집약 업종의 구조적 특성**

K사의 사업 모델은 매장 임차에 완전히 의존한다. 300개 매장의 임차료가 연간 2,800억이다. 이것은 영업 비용이 아니라 실질적인 부채다. 사업을 계속하는 한 이 비용은 고정적으로 나간다.

운용리스를 자본화해서 부채로 인식하면 Debt/EBITDA가 1.8배에서 5.2배로 뛴다. 이자보상배율도 8.2배에서 2.1배로 급감한다. 더 이상 여유 있는 기업이 아니다.

이것이 항공사, 유통업체, 외식 프랜차이즈 등 임차 집약 업종을 분석할 때 특별히 주의해야 하는 이유다. 재무상태표의 부채비율이 실질을 심각하게 과소평가할 수 있다.

---

**함정 — 업종별 기준이 다르다**

유통업이나 항공업을 제조업과 같은 부채비율 기준으로 평가하면 안 된다. 임차 집약 업종은 리스 의무를 포함한 조정순부채로 비교해야 한다.

K사의 부채비율 92%는 제조업 기준으로 보면 낮다. 그러나 유통업 특성을 고려한 조정 부채비율 201%와 Debt/EBITDA 5.2배는 경계할 수준이다.

업황이 나빠져 매출이 10% 감소하면 어떻게 되는가. 임차료는 고정비로 그대로 나간다. 영업이익이 급감한다. 이자보상배율이 더 떨어진다. 차입금 상환 능력이 흔들린다.

---

**결말**

2년 후, 소비 경기 둔화가 왔다. K사의 동일매장 매출이 -12% 하락했다. 임차료 2,800억은 그대로 나갔다. 영업이익이 전년 대비 -58% 급감했다. 이자보상배율은 8.2배에서 1.1배로 추락했다.

K사는 100개 매장을 폐쇄하고 임차 조건을 재협상했다. 그러나 이미 체결된 장기 임차 계약의 위약금만 380억이었다. 유상증자 1,500억 공시가 나왔다. 운영자금 목적이었다.

주가는 고점 대비 -44% 하락했다. 부채비율 92%라는 숫자를 믿고 투자한 사람들이 가장 늦게 위험을 인식했다.

---

**교훈**

> 임차 집약 업종(유통·항공·외식)은 부채비율 대신 조정순부채와 Debt/EBITDA를 써라.
> 리스 주석에서 미인식 단기리스 미래 지급액을 찾아 조정순부채에 더하라.
> 이자보상배율이 2배 미만이면 — 업황 악화 시 생존 능력에 의문이 생긴다.
> K사는 부채비율이 낮은 기업이 아니었다. 부채가 보이지 않는 기업이었다.

### ⑤ DART 실전 박스

**K-IFRS vs US GAAP 차이**

K-IFRS 16과 ASC 842는 거의 동일. 단기리스(12개월 이하) 및 소액자산 리스 면제 가능. 기업이 면제를 얼마나 활용했는지 주석에서 확인 필수.

**DART에서 이렇게 확인한다**

```
Step 1. 주석 → [리스] 항목
        ROU 자산 잔액, 리스부채 잔액 확인
        리스부채 만기별 현금흐름 테이블 확인 (1년 이내 / 1~5년 / 5년 초과)
        단기리스 및 소액 리스 비용 금액 확인

Step 2. 조정순부채 계산
        차입금 + 리스부채(재무상태표) + 단기리스 현재가치(추정) - 현금

Step 3. 이자보상배율 계산
        EBIT(영업이익) / 이자비용
        → 금융비용 전체 사용 (이자비용 + 리스이자비용 포함)

Step 4. 만기 구조 확인
        주석 → [차입금] 만기별 상환 일정
        내년 만기 차입금 / 총차입금 = 단기 차환 필요 비율
        → 40% 초과 시 차환 리스크 집중 분석

Step 5. Debt/EBITDA 계산
        EBITDA = 영업이익 + 감가상각비 + 리스감가상각
        조정순부채 / EBITDA = Debt/EBITDA
```

### 📊 내재가치 체크리스트

**Step 1 — 조정순부채 계산**
```
재무상태표 차입금:         (    )억
+ 리스부채:               (    )억
+ 단기리스 미래지급액 추정: (    )억  (주석에서 확인)
+ 연금 적립부족 (3.3):    (    )억
- 현금 및 현금성자산:     (    )억
─────────────────────────────
조정순부채:               (    )억
조정 부채비율:            (    )%
```

**Step 2 — 생존 능력 3대 지표**
```
① 이자보상배율 = 영업이익 / 금융비용 = (    )배
   판정: 3배↑ 안정 / 1.5~3배 주의 / 1.5배↓ 경고

② 유동비율 = 유동자산 / 유동부채 = (    )%
   판정: 150%↑ 안정 / 100~150% 보통 / 100%↓ 경고

③ Debt/EBITDA = 조정순부채 / EBITDA = (    )배
   판정: 3배↓ 안정 / 3~5배 주의 / 5배↑ 경고
```

**Step 3 — 차환 위험 시뮬레이션**
```

→ 양수 = 차환 또는 추가 조달 필요

```
