# 3.2 장기부채·리스·유동성: 레버리지의 실질과 생존능력

2020년 3월의 기억은 선명하다.

뉴스 알림. "이스타항공, 법정관리 신청."

나는 그 뉴스를 보며 재무제표를 떠올렸다. 이스타항공의 부채비율은 180%였다. 항공업 평균 수준이었다. 그런데 전체 차입금의 72%가 1년 내 만기였고, 이자보상배율은 0.8배였다. 코로나가 터지자 차환이 막혔다. 유동비율 67%.

3개월 만에 끝났다.

대한항공은 살아남았다. 차입금의 80%가 3년 이상 장기였기 때문이다.

같은 하늘을 날았다. 같은 위기를 맞았다. 하나는 살았고 하나는 죽었다. 차이는 부채의 크기가 아니었다. **만기 구조**였다.

기업이 망하는 것은 부채가 많아서가 아니다. **만기에 갚을 현금이 없어서**다.

---

두 기업이 있다. 둘 다 부채비율 120%.

A기업. 차입금 대부분이 5년 이상 장기. 이자보상배율 8배. 유동비율 180%.

B기업. 차입금 중 65%가 내년에 만기. 이자보상배율 1.4배. 유동비율 88%.

같은 부채비율 120%지만 밤에 잠이 오지 않는 기업은 B다. 레버리지 분석에서 중요한 것은 두 가지 축이다.

첫째, **레버리지의 규모** — 부채가 얼마나 되는가. 부채비율이 이것을 말한다.

둘째, **레버리지의 질** — 그 부채를 감당할 수 있는가. 이자보상배율, 유동비율, 만기 구조가 이것을 말한다.

부채비율만 보고 안전을 판단하는 것은 첫 번째 축만 보는 것이다. 두 번째 축 없이는 기업이 실제로 채무를 이행할 능력이 있는지 알 수 없다.

그리고 여기에 하나가 더 있다. **리스 부채**. 재무상태표에 보이는 차입금 외에 운용리스 의무까지 포함해야 진짜 레버리지가 보인다.

이 구조가 실전에서 어떻게 작동하는지, K사의 숫자로 들여다보자.

### K사 — 부채비율 92%의 유통 대기업

친구가 카톡을 보내왔다. 'K사 배당수익률 4.2%다. 부채비율도 92%밖에 안 되고.' 나는 '괜찮아 보이네'라고 답했다. 그리고 습관적으로 DART를 열었다.

부채비율 92%.
배당수익률 4.2%.
안정적 현금흐름.

'배당 안전주'로 딱이었다. 부채비율 100% 미만이니 재무도 탄탄하고.

K사를 관심 종목에 넣고 사업보고서를 열었다. 코스피 대형 유통기업. 전국에 300개 이상의 매장을 운영한다. 매장은 모두 임차.

재무상태표를 열었다. 부채비율 92%. 좋다.

그런데 습관적으로 리스 주석까지 내려갔다. 그 순간 숫자가 달라졌다.

| 항목 | 재무상태표 기준 | 리스 조정 후 |
|---|---|---|
| 총차입금 | 4,200억 | 4,200억 |
| **리스부채 (재무상태표 반영)** | — | **2,100억** |
| **단기리스 미래 지급액 (주석)** | — | **4,700억** |
| 현금 및 현금성자산 | -1,800억 | -1,800억 |
| **조정순부채** | **2,400억** | **9,200억** |
| 자본 총계 | 4,560억 | 4,560억 |
| **부채비율** | **92%** | **조정 후 201%** |
| **Debt/EBITDA** | **1.8배** | **조정 후 5.2배** |
| 이자보상배율 | 8.2배 | 2.1배 |

92%가 201%가 됐다.
Debt/EBITDA가 1.8배에서 5.2배로 뛰었다.
이자보상배율은 8.2배에서 2.1배로 급감했다.

단기리스 면제를 활용한 미래 지급액 4,700억이 리스부채에 잡히지 않고 있었다.

K사는 임차 매장의 상당수를 1년 단위로 계약을 갱신하는 방식을 썼다. K-IFRS 16에서 12개월 이하 리스는 단기리스로 분류해 부채 인식을 면제할 수 있다. K사가 이 면제를 광범위하게 활용하면서 실질적인 임차 의무 중 상당 부분이 재무상태표 밖에 있었다.

'부채비율 92%라는 숫자를 믿을 뻔했다.'

### 유효이자율법과 리스 회계 — K사를 해부하기 위한 도구

K사의 숫자를 제대로 읽으려면 두 가지 회계 구조를 이해해야 한다.

기업이 채권(사채)을 발행할 때 시장 이자율과 표면 이자율이 다르면 할인 또는 할증 발행이 일어난다.

```
액면가 100억, 표면이자율 3%, 시장이자율 5%인 경우:
  발행가(현재가치) = 3억/1.05 + 3억/1.05² + ... + 103억/1.05³
                  = 94.5억 (할인 발행)
  사채할인발행차금 = 5.5억

매 기간 이자비용 계산 (유효이자율법):
  이자비용 = 장부가액 × 시장이자율(5%)
  현금 지급 = 액면가 × 표면이자율(3%)
  할인차금 상각 = 이자비용 - 현금지급 → 장부가액 증가
```

할인 발행된 채권의 장부가액은 만기에 가까워질수록 액면가로 증가한다. 이자비용은 현금 지급보다 크다. 손익계산서의 이자비용이 실제 현금 지출 이자와 다를 수 있다. 이것이 FCF 계산에 영향을 준다. 할인 발행 사채가 있는 기업에서 비현금 이자비용(할인차금 상각)이 CFO에 가산 조정되므로, 이를 모르면 CFO가 과대평가된 것으로 착각할 수 있다.

리스 회계는 K사의 숫자를 뒤집는 핵심이었다. ASC 842 / K-IFRS 16 이전에는 운용리스의 미래 지급 의무가 재무상태표에 올라오지 않았다. 지금은 다르다. 12개월을 초과하는 리스는 사용권자산(ROU, 리스로 쓸 권리를 자산으로 인식한 것)과 리스부채로 재무상태표에 올라온다.

```
리스 개시 시:
  ROU 자산 ↑   /  리스 부채 ↑ (동일 금액)

매 기간:
  리스 이자비용 = 리스부채 × 내재이자율
  감가상각비 = ROU 자산 / 리스기간

금융리스 vs 운용리스 (리이용자 기준):
  금융리스: 이자비용 + 감가상각비로 분리 → 초기 비용 많음
  운용리스: 단순 리스료로 처리 → 기간 균등 비용
```

핵심은 이것이다. 운용리스로 분류된 임차 계약도 이제 대부분 재무상태표에 올라온다. 그런데 **단기리스(12개월 이하)** 와 **소액자산 리스**는 면제 선택이 가능하다. K사처럼 이 면제를 광범위하게 활용하면 리스 부채 규모가 실질보다 낮게 표시된다.

탐지법: 주석에서 '단기리스 비용'과 '리스부채 잔액'을 비교하라. 단기리스 비용 / (리스부채 + 단기리스 비용 현재가치) 비율이 50%를 넘으면, 기업이 단기 계약 갱신 방식으로 부채 인식을 회피하고 있을 가능성이 높다.

### 조정순부채 — 진짜 레버리지 계산법

재무상태표에 보이는 차입금만으로 순부채를 계산하면 부족하다.

```
조정순부채 계산:

재무상태표 차입금 합계:             (    )억
+ 리스부채 (재무상태표):            (    )억
+ 단기리스 미래 지급액 (주석):      (    )억  ← 면제 활용분
+ 연금 적립 부족분 (주석):          (    )억  ← 3.3에서 상세
- 현금 및 현금성자산:              (    )억
──────────────────────────────────
조정순부채:                        (    )억
```

K사의 사업 모델은 매장 임차에 완전히 의존한다. 300개 매장의 임차료가 연간 2,800억이다. 이것은 영업 비용이 아니라 실질적인 부채다. 사업을 계속하는 한 이 비용은 고정적으로 나간다.

운용리스를 자본화해서 부채로 인식하면 Debt/EBITDA가 1.8배에서 5.2배로 뛴다. 이자보상배율도 8.2배에서 2.1배로 급감한다. 더 이상 여유 있는 기업이 아니다.

이것이 항공사, 유통업체, 외식 프랜차이즈 등 임차 집약 업종을 분석할 때 특별히 주의해야 하는 이유다. 재무상태표의 부채비율이 실질을 심각하게 과소평가할 수 있다.

### 이자보상배율과 유동비율 — 생존 능력의 두 지표

**이자보상배율(Interest Coverage Ratio)**은 영업이익으로 이자를 몇 번 낼 수 있는지를 측정한다.

```
이자보상배율 = EBIT / 이자비용

판정 기준:
  3배 이상  → 안정적
  1.5~3배  → 주의 구간
  1.5배 미만 → 이자 납부 능력 한계선 근접
  1배 미만  → 영업이익만으로 이자도 못 냄
```

**유동비율(Current Ratio)**은 1년 내 만기 부채를 1년 내 현금화 가능한 자산으로 갚을 수 있는지를 측정한다.

```
유동비율 = 유동자산 / 유동부채

판정 기준:
  150% 이상 → 안정적
  100~150% → 보통
  100% 미만 → 단기 지급 능력 부족
```

그런데 유동비율에는 함정이 있다. 재고자산이 당장 현금화되기 어려운 기업이라면 재고를 제외한 당좌비율이 더 의미 있다.

```
당좌비율 = (유동자산 - 재고자산) / 유동부채
```

2년 후, 소비 경기 둔화가 왔다.

K사의 동일매장 매출이 -12% 하락했다. 임차료 2,800억은 그대로 나갔다. 매출이 줄어도 임차료는 줄지 않는다. 그것이 고정 부채의 본질이다.

영업이익이 전년 대비 -58% 급감했다.
이자보상배율은 8.2배에서 1.1배로 추락했다.

K사는 100개 매장을 폐쇄하고 임차 조건을 재협상했다. 그러나 이미 체결된 장기 임차 계약의 위약금만 380억이었다.

유상증자 1,500억 공시. 운영자금 목적.

주가는 고점 대비 -44% 하락했다. 부채비율 92%라는 숫자를 믿고 투자한 사람들이 가장 늦게 위험을 인식했다.

K사는 부채비율이 낮은 기업이 아니었다. **부채가 보이지 않는 기업**이었다.

나는 리스 주석을 열어본 덕에 매수하지 않았다. 92%라는 숫자 뒤에 201%가 숨어 있었다. 그 차이를 아는 것과 모르는 것의 차이가 -44%였다.

### 실전: DART에서 확인하는 법

**K-IFRS vs US GAAP 차이**

K-IFRS 16과 ASC 842는 거의 동일. 단기리스(12개월 이하) 및 소액자산 리스 면제 가능. 기업이 면제를 얼마나 활용했는지 주석에서 확인 필수.

```
Step 1. 주석 → [리스] 항목
        ROU 자산 잔액, 리스부채 잔액 확인
        리스부채 만기별 현금흐름 테이블 확인 (1년 이내 / 1~5년 / 5년 초과)
        단기리스 및 소액 리스 비용 금액 확인

Step 2. 조정순부채 계산
        차입금 + 리스부채(재무상태표) + 단기리스 현재가치(추정) - 현금

Step 3. 이자보상배율 계산
        EBIT(영업이익) / 이자비용
        → 금융비용 전체 사용 (이자비용 + 리스이자비용 포함)

Step 4. 만기 구조 확인
        주석 → [차입금] 만기별 상환 일정
        내년 만기 차입금 / 총차입금 = 단기 차환 필요 비율
        → 40% 초과 시 차환 리스크 집중 분석

Step 5. Debt/EBITDA 계산
        EBITDA = 영업이익 + 감가상각비 + 리스감가상각
        조정순부채 / EBITDA = Debt/EBITDA
```

**조정순부채 계산**
