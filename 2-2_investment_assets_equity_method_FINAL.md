# 2.2 투자자산: 지분법 기업이 숨기고 있는 것들

---

## "버튼 하나"

유튜브에서 추천 종목으로 올라온 기업이었다. 코스피 중견 그룹의 지주회사 F사.

DART에 접속해서 F사를 검색했다. 두 달째 추적하던 종목이었다. 별도 사업보고서를 클릭했다. PDF가 열렸다. 재무상태표, 첫 줄.

부채비율 38%.

무차입에 가까운 수준이었다. 스크롤을 내렸다.

현금성 자산 2,400억.
순현금 포지션.
3년 연속 안정적인 배당.
PBR 0.65배.

심장이 빨라졌다. 저평가된 우량 지주회사. 안전하고, 싸다. 이런 종목이 왜 아직 이 가격에 있는가. 시장이 모르는 건가.

매수 버튼 위에 손을 올렸다.

그때 DART 화면 상단에 작은 버튼이 보였다. **[연결]**. 연결 사업보고서로 전환하는 버튼이었다. 습관적으로 눌렀다.

3초.

화면의 숫자들이 완전히 바뀌는 데 3초가 걸렸다.

손이 멈췄다. 뭔가 이상했다. 아니, 이상한 정도가 아니었다.

---

### 바닥이 빠지는 순간

부채비율이 38%에서 287%로 바뀌었다.

한 줄 아래. 순현금 +1,800억이 순부채 -4,200억으로 바뀌었다.

그 아래. 영업이익 820억 흑자가 -140억 적자로 바뀌었다.

| 항목 | 별도 재무제표 | 연결 재무제표 | 차이 |
|---|---|---|---|
| **부채비율** | **38%** | **287%** | **+249%p** |
| **순부채/순현금** | **순현금 +1,800억** | **순부채 -4,200억** | **6,000억 차이** |
| **영업이익** | **820억** | **-140억** | **960억 차이** |
| 특수관계자 매출 비중 | — | 61% | — |

같은 기업이다. 같은 날짜의 재무제표다. 버튼 하나만 눌렀다.

나는 화면 앞에서 멈췄다. 순현금 1,800억짜리 안전한 기업이, 순부채 4,200억짜리 기업이 됐다. 영업이익 820억 흑자가 140억 적자가 됐다. 같은 회사를 보고 있는 게 맞는가.

바뀐 것은 기업이 아니었다. 내가 보는 창이 바뀐 것이었다.

나는 매수 버튼에서 손을 뗐다. 이 간극의 정체를 알아야 했다.

### 이익은 끌어오고, 부채는 감추는 구조

답은 **지분법**이라는 회계 구조에 있었다.

F사는 계열사 지분 35%를 보유하고 있었다. 그 계열사의 차입금이 얼마인지를 확인했다.

6,800억.

부채비율 420%.

이 6,800억은 F사 재무상태표에 **한 줄도 나오지 않았다.** 대신 '관계기업투자주식 1,200억'이라는 자산 항목 하나만 올라와 있었다.

6,800억짜리 폭탄이 1,200억짜리 자산 한 줄 뒤에 숨어 있었다.

왜 이런 일이 가능한가.

기업이 보유한 다른 회사의 주식이나 채권은 **보유 목적과 비율**에 따라 분류된다. 분류에 따라 평가 방법이 달라지고, 평가손익이 어디로 가는지가 달라진다.

| 분류 | 보유 비율/목적 | 평가 방법 | 평가손익 귀속 |
|---|---|---|---|
| **단기매매(FVPL)** | 단기 차익 목적 | 공정가치 | 당기손익(NI) |
| **FVOCI(지분상품)** | 장기 보유 선택 | 공정가치 | OCI (영구) |
| **상각후원가** | 만기 보유 채무증권 | 상각후원가 | 이자수익만 NI |
| **지분법** | 유의적 영향력 20~50% | 지분법 | NI (지분 손익) |
| **연결** | 지배력 50%+ | 완전 연결 | 전체 반영 |

투자자 입장에서 가장 중요한 구분은 두 가지다.

**FVPL(단기매매)**: 보유 주식의 공정가치 변동이 매 분기 손익계산서를 직접 건드린다. 주가가 오르면 이익, 내리면 손실. 변동성이 크다.

**지분법**: 피투자회사의 손익 중 지분율만큼 모회사 손익에 반영된다. 피투자회사가 100억 이익을 내면, 35% 지분 보유 기업은 35억을 지분법이익으로 인식한다.

여기까지는 합리적으로 보인다. 문제는 그 다음이다.

### 140억은 진짜 돈인가

지분법의 회계 처리를 단계별로 따라가 봤다.

```
투자 시점:
  관계기업투자주식 1,200억 (취득원가)

이후 매 기간:
  피투자기업 당기순이익 400억 × 지분율 35% = 지분법이익 140억
  → 관계기업투자주식 +140억 / 지분법이익 +140억 (손익계산서 반영)

배당 수령:
  피투자기업 배당 200억 × 35% = 70억 수령
  → 현금 +70억 / 관계기업투자주식 -70억 (자산 감소)
  → 배당은 이미 지분법이익으로 인식한 이익의 회수이므로 추가 손익 없음
```

멈췄다. 다시 읽었다.

지분법이익 140억은 손익계산서에 올라온다. 그런데 이 140억은 피투자회사가 배당을 하든 안 하든 인식된다. 실제로 현금이 들어오는 것은 배당 70억뿐이다.

나머지 70억.

140억 빼기 70억. 그 차이는 장부상의 이익이다. 통장에 찍히지 않는 이익.

**지분법이익과 배당수익의 차이**가 크다는 것은 무엇을 의미하는가. 피투자기업이 이익을 내고 있지만 배당으로 현금을 돌려보내지 않는다는 뜻이다. 피투자기업에 현금이 묶여 있거나, 피투자기업의 이익 자체가 현금흐름을 수반하지 않는 회계적 이익일 수 있다.

시장은 지분법이익을 영업이익과 같은 질(quality)로 평가하는 경향이 있다. 그러나 지분법이익의 상당 부분은 현금 유입을 수반하지 않는다. 배당으로 실현되지 않은 지분법이익에 PER을 적용하면 기업가치가 과대평가된다.

### 4,200억은 어디에 숨어 있었나

다시 F사의 재무제표로 돌아갔다. 이번에는 피투자기업의 숫자를 직접 들여다봤다.

```
피투자기업 재무상태표:
  총자산 5,000억
  총부채 4,200억 ← 이 부채는 모회사 재무상태표에 직접 나타나지 않음
  순자산   800억

모회사 재무상태표에 반영되는 것:
  관계기업투자주식 = 800억 × 35% = 280억만 자산으로 올라옴
  피투자기업의 4,200억 부채는 보이지 않음
```

연결 재무제표라면 자회사의 자산과 부채가 모두 모회사에 합산된다. 그런데 지분법에서는 피투자기업의 장부 전체가 아니라 **순자산 중 지분율만큼**만 모회사 자산에 반영된다.

4,200억의 부채가 280억짜리 자산 한 줄 뒤에서 사라진 것이다.

무슨 일이 일어나는가. 피투자기업의 부채가 문제가 되면, 모회사의 장부에는 나타나지 않지만 실질적인 리스크는 모회사가 부담할 수 있다. 피투자기업이 부채를 갚지 못해 기업 가치가 하락하면, 관계기업투자주식 280억을 손상처리해야 한다. 최악의 경우 피투자기업이 디폴트 상황에 빠지면 모회사가 보증을 선 경우 우발부채로 현실화된다.

F사에서 바로 이 구조가 작동하고 있었다.

### 별도 이익 820억의 정체

밥을 먹다 말고 다시 노트북을 열었다. F사의 별도 영업이익 820억이 어디서 오는지를 해부했다.

```
F사 별도 영업이익 820억 구성:
  계열사 배당금 수익:  680억 (83%)
  자체 사업 영업이익:  140억 (17%)
```

83%.

별도 영업이익의 83%가 계열사 배당금이었다. F사가 자기 힘으로 번 돈은 140억뿐이었다.

그 배당금을 준 계열사들의 속사정은 어떠한가. 연결 재무제표를 열었다. 핵심 계열사들의 차입금 합계가 수조 원이었다. 이 계열사들은 차입금으로 운영하면서 배당을 지급하고 있었다. 빚으로 배당을 주고 있었던 것이다.

그 배당이 F사의 별도 이익처럼 보였다.

연결 재무제표에서 이 계열사들의 부채가 고스란히 반영되자 부채비율이 38%에서 287%로 뛰었다. 계열사 영업 손실이 합산되자 연결 영업이익은 적자로 전환됐다.

별도 재무제표는 지주회사가 계열사로부터 무엇을 받는지를 보여준다. 연결 재무제표는 그룹 전체가 실제로 어떤 상태인지를 보여준다. 지주회사 분석에서 별도만 보는 것은, 배당을 받는 손만 보고 그 돈이 어디서 어떻게 만들어지는지를 보지 않는 것과 같다.

DART에서 검색하면 기본으로 뜨는 것이 연결 사업보고서다. 그런데 연결 재무제표에는 계열사들이 합산돼 있어서 지주회사 자체의 상태를 파악하기 어렵다는 이유로, 많은 투자자가 별도 재무제표를 '더 순수한 기업 본체'로 여긴다.

그 판단이 지주회사에서는 역효과를 낸다. 지주회사의 '본체'는 결국 그룹 전체다. 계열사들이 무너지면 지주회사의 배당 수익도 사라진다.

별도 재무제표의 견고함은 계열사들이 계속 배당을 줄 수 있을 때만 유효하다. 그 전제가 무너지는 시나리오를 연결 재무제표에서 확인해야 한다.

### 그 뒤에 벌어진 일

3년 후의 이야기다.

어느 날 새벽, HTS를 열었다. F사의 그룹 내 핵심 계열사 두 곳의 실적 경고 공시가 떠 있었다. 업황 악화. 과도한 차입금으로 버텨온 계열사들이 더 이상 버티지 못했다.

계열사 배당이 중단됐다.

F사의 별도 영업이익이 820억에서 180억으로 급감했다.

다음 분기. 관계기업투자주식에 대한 대규모 손상차손이 발생했다. 수천억의 자산이 한 분기 만에 사라졌다.

별도 재무제표 기준 '순현금 안전 기업'이었던 F사는 3년 만에 자금 조달에 나서야 했다.

주가: 고점 대비 **-62%**.

그날 밤, 화면의 -62%를 보면서 생각했다. 3년 전 그 저녁, DART에서 [연결] 버튼을 눌렀을 때 이미 답은 나와 있었다.

처음부터 연결 재무제표를 봤던 투자자는 이 그룹이 보여주는 것과 실제 사이의 간극을 알고 있었다. 별도만 보고 매수한 투자자는 반토막보다 더 깊은 하락을 경험했다.

지주회사는 반드시 연결 재무제표를 기준으로 분석하라. DART에서 연결과 별도, 두 개를 나란히 열어라. 부채비율 차이가 100%p를 넘는다면 — 별도 재무제표의 안전함은 절반의 진실이다. 지주회사 밸류에이션은 부문별 합산 평가(Sum-of-the-Parts, SOTP)를 기본으로 하되, 각 자회사의 가치에서 순부채를 차감하고 지주회사 할인율 20~40%를 적용하라.

### 실전: DART에서 확인하는 법

**K-IFRS vs US GAAP 차이**

K-IFRS 9에서 지분상품 FVOCI 선택 시 매도해도 손익을 NI로 전환할 수 없다(영구 OCI). US GAAP에서는 AFS 지분상품 처분 시 손익을 NI로 재분류한다. 한국 기업에서 FVOCI 선택한 주식 보유 규모가 크면, 처분해도 이익으로 잡히지 않는다는 점에 주의하라.

지분법 회계는 K-IFRS와 US GAAP이 거의 동일하게 적용된다.

```
Step 1. DART에서 두 개의 보고서를 나란히 열어라
        연결 사업보고서 / 별도 사업보고서
        부채비율, 영업이익, 순현금 각각 비교

Step 2. 별도 사업보고서 → 주석 → [관계기업 투자]
        지분법 적용 기업 목록 확인
        각 피투자기업의 주요 재무정보(자산·부채·매출·이익) 확인

Step 3. 별도 손익계산서 → 영업이익 구성 분해
        배당금수익, 지분법이익이 영업이익에서 차지하는 비중 계산
        → 50% 초과 시 자체 영업 수익성 별도 계산

Step 4. 연결 주석 → [특수관계자 거래]
        특수관계자 매출 금액, 전체 매출 대비 비중 확인
        → 60% 초과 시 외부 시장 경쟁력 재평가 필요
```

### 실전 체크리스트

**별도 vs 연결 핵심 3대 지표 비교**
```
DART에서 두 보고서를 나란히 열고 직접 채워라:

                    별도      연결      차이
부채비율(%)        ____      ____      ____
순부채/현금(억)    ____      ____      ____
영업이익(억)       ____      ____      ____

판정 기준:
  부채비율 차이 100%p 초과 → 연결 기준으로만 분석
  영업이익 별도 > 연결      → 계열사 손실 흡수 중
  별도 순현금 + 연결 순부채 → 지분법 뒤에 부채 숨겨짐
```

**지분법 의존도 계산**
```
별도 손익계산서에서:
  배당금수익 + 지분법이익 합계 = (    )억
  별도 영업이익               = (    )억
  지분법 의존도               = (    )%

판정 기준:
  30% 미만  → 자체 수익 기반 견고
  30~60%   → 관계기업 실적 모니터링 필요
  60% 초과  → 이 기업의 이익은 계열사 건강에 달려 있다
              → 즉시 피투자기업 재무 건전성 분석으로 이동
```

**피투자기업 재무 건전성 직접 확인**
```
DART: 별도 사업보고서 → 주석 → [관계기업 투자] 항목

피투자기업별 주요 재무정보 표에서:
  ① 총자산, 총부채 → 부채비율 직접 계산
  ② 당기순이익 → 수익성 확인
  ③ 배당금 지급 여부 → 모회사 배당수익의 지속가능성

판정 기준:
  피투자기업 부채비율 200% 초과 → 레버리지 리스크 경고
  피투자기업 부채비율 400% 초과 → 관계기업투자주식 손상 가능성 높음
  피투자기업 순손실 + 배당 지급  → 차입금으로 배당, 지속 불가능
```

**지분법이익 vs 배당수익 괴리 계산**
```
지분법이익   = (    )억  ← 손익계산서 반영
배당수익     = (    )억  ← 실제 현금 유입
괴리         = (    )억

괴리가 크다는 것 = 피투자기업이 이익은 내지만 현금은 쌓아두고 있다
                  또는 피투자기업 이익 자체가 현금을 수반하지 않는다
→ 괴리가 지분법이익의 50% 초과 시 피투자기업 현금흐름 별도 확인
```
